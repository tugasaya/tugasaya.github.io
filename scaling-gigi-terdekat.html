18. Diketahui : roda gigi a mempunyai jumlah gigi = 20 gigi, dan roda gigi b = 30 gigi, roda gigi c = 30 gigi, jika roda gigi c berputar 240 rpm, maka kecepatan sudut roda gigi b adalah?


Kecepatan sudut roda gigi B adalah 160 Rpm.

Penjelasan dengan langkah-langkah :

Korelasi adalah ukuran dari seberapa dekat dua variabel berubah dalam hubungan satu sama lain. Dalam konteks teknik analisis, korelasi kerap digunakan untuk mencari suatu hubungan diantara dua variabel yang juga memiliki sifat kuantitatif.

Diketahui :
Roda gigi A = 20 gigi

Roda gigi B = 30 gigi

Roda gigi C = 30 gigi

Gigi C berputar di 240 Rpm

Ditanya : Kecepatan sudut roda gigi B adalah ?

Dijawab :

Perbandingan jumlah gigi ketiga roda adalah

[tex]nA : nB : nC = 20:30:30 = 2:3:3[/tex]

Roda C dan A seporos maka ωA = ωC = 240 Rpm

Perbandingan jumlah roda A dan B =  [tex]\frac{2}{3}[/tex], artinya saat roda B berputar 2 kali, roda A telah berputar sebanyak 3 kali.

ωB = ([tex]\frac{2}{3}[/tex]) ωA

ωB =  [tex]\frac{2}{3} x 240 Rpm[/tex]

ωB = [tex]160 Rpm[/tex]

Jadi, kecepatan sudut roda gigi B adalah 160 Rpm.
