10. jelaskan perbedaan parent proxy dengan sibling proxy


perbedaannya
Parent adalah type proxy yang mempunyai kewajiban mengambil object dari internet dan menyerahkan ke client apabila object tersebut belum ada pada disk cache, apabila object yang di minta sudah ada dalam memori atau di disk cache, maka object akan langsung di serahkan ke client.

Sibling adalah type proxy yang hanya mempunyai kewajiban menyerahkan object , jika object yang diminta ada dalam memori atau disk cache proxy

21. domain pada internet-CO-ID-COM-EDU-ORG-HTML-AU-ONIONJawab Plis​


Domain pada internet

".co" domain untuk beberapa negara. ( Jika di Indonesia menggunakan .co.id, namun negara kolombia menggunakan langsung .co )

".id" domain untuk negara Indonesia

".com" domain internasional

".edu" domain untuk pendidikan

".org" domain untuk organisasi

".html" domain untuk tipe ekstensi yang berbentuk html, domain ini juga bisa berupa file dan bisa dijalankan offline

".au" domain untuk negara australia

".onion" domain untuk pengguna aplikasi khusus ( Dalam hal ini butuh aplikasi yang bernama TOR ) terkadang disebut domain untuk dark web

Pembahasan

Yang kita liat diatas ialah top level domain, domain diatas berarti domain yang tidak bisa ditambahkan lagi. Sebagai contoh domain pada web brainly.co.id ialah ".co.id", ".co" adalah second level domain sedangkan ".id" adalah top level domain.

Diatas juga merupakan domain yang bisa digunakan sebagai website dan gmail ( Kecuali .html dan .onion ). Fungsi domain cukup penting sebagai identitas suatu organisasi ataupun website.

Jawaban:

a. high anonymity proxy

Penjelasan:

Proxy server adalah server yang bertindak sebagai perantara antara client dan server tujuan. Saat client mengakses sebuah situs web, alamat IP client akan terlihat oleh server tujuan. Namun, jika client menggunakan proxy server, maka alamat IP yang terlihat oleh server tujuan adalah alamat IP proxy server, bukan alamat IP client.

Ada beberapa jenis proxy server, yaitu:

Transparent proxy serverAnonymous proxyDistorting proxy:High anonymity proxyFitur filtering server

30. apa itu proxy jelaskan


Penjelasan proxy

Pengertian proxy adalah server yang menyediakan suatu layanan untuk meneruskan setiap permintaan user kepada server lain yang terdapat di internet. Atau definisi proxy server yang lainnya yaitu suatu server atau program komputer yang mempunyai peran sebagai penghubung antara suatu komputer dengan internet.

proxy adalah suatu server yang menyediakan layanan untuk meneruskan setiap permintaan kita kepada server lain di internet. Denganproxy, maka identitas komputer anda berupa IP menjadi tersembunyi dikarenakan yang dikenali server yang direquest adalah IP dari server proxy anda. Proxy ini pada umumnya digunakan untuk kegiatan menyembunyikan identitas atau untuk menghindari pemblokiran akses ke suatu server.
