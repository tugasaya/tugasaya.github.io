Tabunagn Dodo = Rp. 1.500.000,00

Tabungan Didi = Rp. 1.000.000,00

Harga 1 set sepatu = Rp. 250.000,00

Sisa uang Didi = ?

Sisa uang Dodo = ?

---

Sisa uang Didi :

= jumlah tabunagn Didi - harga 1 set sepatu

= Rp. 1.000.000,00 - Rp. 250.000,00

= Rp. 750.000,00

Sisa uang Dodo :

= jumlah tabungan Dodo - harga 1 set sepatu

= Rp. 1.500.000,00 - Rp. 250.000,00

= Rp. 1.250.000,00

Total seluruh uang yang dikeluarkan oleh Didi dan Dodo :

= harga 1 set sepatu × 2

= Rp. 250.000,00 × 2

= Rp. 500.000,00

Jawab:Sisa tabungan Dodo

→ Rp. 1.500.000,00
