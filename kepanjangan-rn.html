panjang rusuk K,N = L,M

jadi, panjang rusuk dari R,N adalah 8 cm

27. berikut ini berkaitan dengan pola bilangan persegi panjanga. tentukan suku ke-16 Dan suku ke-19b. jika Rn=552 suku ke berapakah Rn​


Jawaban:

a. Suku ke - 16 = 257

Suku ke-19 = 362

b. Rn = 23

Penjelasan dengan langkah-langkah:

a. Suku ke - 16 :

Un = n.n + 1

U16 = 16.16 + 1

= 256 + 1

= 257

Suku ke-19 :

Un = n.n + 1

U19 = 19.19 + 1

= 361 + 1

= 362

b. Un = n.n + 1

Rn = n.n + 1

552 = n² + n

n² + n - 552 = 0

(n + 24) (n - 23)

n = 23
