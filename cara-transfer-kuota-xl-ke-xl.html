4. Nilai x yang memenuhi persamaan l2-xl²-l2-xl=3l2-xl+5 adalah​


Penjelasan dengan langkah-langkah:

|2 - x|² - |2 - x| = 3|2 - x| + 5

misal |2 - x| = p, maka :

p² - p = 3p + 5

p² - p - 3p - 5 = 0

p² - 4p - 5 = 0

(p + 1)(p - 5) = 0

p = -1 atau p = 5

• untuk p = -1

|2 - x| = -1 (tidak memenuhi, karena nilai mutlak tidak mungkin bernilai negatif)

• untuk p = 5

|2 - x| = 5

2 - x = -5 atau 2 - x = 5

-x = -5 - 2 atau -x = 5 - 2

-x = -7 atau -x = 3

x = 7 atau x = -3

Jadi, HP = {-3, 7}

25. himpunan penyelesaian dari l7-xl ≤ 2 adalah ... ( menggunakan cara definisi )


Himpunan penyelesaian dari |7 - x| ≤ 2 adalah

Hp = {x | 5 ≤ x ≤ 9, x ∈ R}

Nilai tesebut diperoleh dari perhitungan himpunanan penyelesaian pertidaksamaan nilai mutlak dengan menggunakan definisi nilai mutlak. Simak pembahasan berikut.

Pembahasan

Himpunan penyelesaian dari pertidaksamaan nilai mutlak |7 - x| ≤ 2 dengan menggunakan metode definisi nilai mutlak.

Jawab:

Nilai mutlak dilambangkan dengan |x| dan difenisikan sebagai berikut:

1. |x| = x, untuk x ≥ 0

2. |x| = -x, untuk x < 0

Sehingga sesuai definisi nilai mutlak himpunan penyelesaian dari |7 - x| ≤ 2 adalah

|7 - x| ≤ 2

7 - x ≤ 2 atau -(7 - x) ≤ 2

untuk 7 - x ≤ 2

7 - x ≤ 2

7 - 2 ≤ x

5 ≤ x

x ≥ 5

untuk -(7 - x) ≤ 2

-(7 - x) ≤ 2

-7 + x ≤ 2

x ≤ 2 + 7

x ≤ 9

Maka diperoleh x ≥ 5 atau x ≤ 9. Karena x terletak antara 5 dan 9, himpunan penyelesaian dari |7 - x| ≤ 2 adalah

Hp = {x | 5 ≤ x ≤ 9, x ∈ R}
