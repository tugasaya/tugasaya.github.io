1. Bagaimanakah cara mencari simpangan baku?


Simpangan baku adalah akar dari varians, artinya seberapa jauh populasi tadi dari rata ratanya 
langkah yang harus dilakukan adalah 
cari rata rata nya dulu [tex]\bar{x}[/tex]
Kemudian rumusnya
misalkan ada 3 data maka penyelesaiannya adlah 
[tex]\sigma = \sqrt{\frac{(x_1-\bar{x})^2+(x_2-\bar{x})^2+(x_3-\bar{x})^2}{3-1}} [/tex]
Simpangan baku atau juga yang sering kita kenal dengan nama deviasi standard (standard deviation) adalah ukuran persebaran data. Simpangan ini bisa diartikan jarak rata-rata penyimpangan antara nilai hasil pengukuran dengan nilai rata-rata . 

Rumus Simpangan Baku untuk Data Tunggal
Jika sobat mempunyai sekumpulan data kuatitatif tunggal (tidak berkelompok) yang dinyatakan oleh x1,x2,x3,….,xn maka dapat dicari simpangan bakunya dengan rumusuntuk data sample menggunakan rumus
untuk data populasi menggunkan rumusRumus Simpangan Baku Untuk Data KelompokMisal sobat punya data kelompok yang dinyatakan dengan x1,x2,x3,…,xn dan masing-masing mempunyai frekuensi fi,f2,f3,…,fn maka simpangan bakunya dapat dicari dengan rumusuntuk sample menggunakan rumusuntuk populasi menggunakan rumus

Rumus letak Qᵢ (kuartil) pada data tunggal :

[tex] \boxed{Letak \: \: Q_i = \frac{i(n + 1)}{4}}[/tex]

Maka :

[tex]Letak \: \: Q_1 = \frac{1 \times (7 + 1)}{4} = \frac{8}{4} = 2[/tex]

[tex]\boxed{\boxed{Q_1 = 3}}[/tex]

[tex]Letak \: \: Q_2 = \frac{2 \times (7 + 1)}{4} = \frac{16}{4} = 4[/tex]

[tex]\boxed{\boxed{Q_2 = 3}}[/tex]

[tex]Letak \: \: Q_3 = \frac{3 \times (7 + 1)}{4} = \frac{24}{4} = 6[/tex]

[tex]\boxed{\boxed{Q_3 = 4}}[/tex]

[tex]H : jangkauan \: \: interkuartil[/tex]

[tex]H = Q_3 - Q_1 = 4 - 3 \: \: > > \: \: \boxed{ \boxed{H = 1}}[/tex]

[tex]Q_d :jangkauan \: \: semi \: \: interkuartil[/tex]

[tex]Q_d = \frac{1}{2} \times H = \frac{1}{2} \times 1 \: \: >> \: \: \boxed{ \boxed{Q_d = \frac{1}{2} }}[/tex]

Rumus letak Dᵢ (desil) pada data tunggal :

[tex] \boxed{Letak \: \: D_i = \frac{i(n + 1)}{10}}[/tex]

Maka :

[tex]Letak \: \: D_3 = \frac{3 \times (7 + 1)}{10} = \frac{24}{10} = 2,4[/tex]

[tex]D_3 = X_2 + (0,4 \times (X_3 - X_2))[/tex]

[tex]D_3 = 3 + (0,4 \times (3 - 3)) = 3 + (0,4 \times 0) \: \: >> \: \: \boxed{ \boxed{D_3 = 3}}[/tex]

Rumus letak Pᵢ (persentil) pada data tunggal :

[tex] \boxed{Letak \: \: D_i = \frac{i(n + 1)}{100}}[/tex]

Maka :

[tex]Letak \: \: P_{24} = \frac{24 \times (7 + 1)}{100} = \frac{192}{100} = 1,92[/tex]

[tex]P_{24} = X_1 + (0,92 \times (X_2 - X_1))[/tex]

[tex]P_{24} = 2 + (0,92 \times (3 - 2)) = 2 + (0,92 \times 1) = 2 + 0,92 \: \: > > \: \: \boxed{ \boxed{P_{24} = 2,92}}[/tex]

[tex]X_r = \frac{2 + 3 + 3 + 3 + 4 + 4 + 5}{7} = \frac{24}{7} = 3,4286[/tex]

[tex] \boxed{SR = \frac{1}{n} \: \sum \limits^{n}_{i = 1} | \: X_i \: - \:X_r\: |}[/tex]

[tex]SR = \frac{1}{7} \: \sum \limits^{7}_{i = 1} | \: X_i \: - \:3,4286\: |[/tex]

[tex]\sf simpangan\ baru = deviasi\ standard = S= \sqrt{S^2}[/tex]

[tex]\sf simpangan\ baru = deviasi\ standard = S= \sqrt{S^2}= \sqrt{42,64} = 65,29[/tex]


7. Cara mencari persentase dari nilai antsra 70 dan 90, dengsn nilai rata-rats 80 dan simpangan baku 5


[tex]X=70\rightarrow z=\dfrac{X-\mu}{\sigma}=\dfrac{70-80}{5}=-2[/tex]
[tex]P(X \leq 70)=P(z \leq -2)=0.028[/tex]

Perhatikan gambar daerah I dan II akan sama besar (simetris) , maka
[tex]P(70 \ \textless \ X \ \textless \ 90)=P(-2\ \textless \ z\ \textless \ 2)=1-0.0228-0.0228=0.9544[/tex]

Jadi persentasi dari nilai antara 70 dan 90 = 95,44%
