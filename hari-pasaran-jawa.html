2. Apa itu JWLK dalam kalimat Jawa?

[tex] \: [/tex]

PENGERTIAN

JWLK adalah pola dalam kalimat bahasa Jawa yang kepanjangannya sebagai berikut:

J = JejerW = WasesaL = LesanK = Keterangan

[tex] \: [/tex]

- Dalam bahasa Indonesia:

Jejer = SubyekWasesa = PredikatLesan = ObjekKeterangan = Keterangan

[tex] \: [/tex]

Pembahasan:

1. Jejer

=> Menunjukkan seseorang

2. Wasesa

=> Menunjukkan kegiatan yang dilakukan

3. Lesan

=> Menunjukkan barang/benda

4. Keterangan

=> Menunjukkan keterangan tempat atau waktu
