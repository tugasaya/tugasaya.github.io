2. Software atau aplikasi yang digunakan dalam membangun server streaming multimedia adalah


STREAMING SERVER

Jawaban

Streaming Server adalah sebuah web server atau aplikasi yang terdapat pada sebuah server, yang berguna untk menjalankan file audio maupun video secara real-time. Streaming server membuat kita memasukkan file – file berupa audio maupun video secara terpisah dari web server yang kita jalankan. Beberapa situs yang menyediakan layanan streaming video, menggunakan streaming server untuk menjalankan layanannya. Contohnya adalah Youtube, Metacafe, dan Dailymotion.

Keuntungan dan Kekurangan Penggunaan Streaming Server

Keuntungan:

Streaming biasanya membutuhkan bandwidth yang lebih sedikit daripada mengunduh biasa, karena pengguna tidak harus mengunduh secara penuh video. Keamanan file lebih terjamin, karena tidak ada data yang tersimpan di dalam komputer pengguna. Streaming ke beberapa klien dapat dilakukan hanya dari satu jalur streaming saja. Pengguna yang menonton video dapat melakukan fast forward atau skip ke beberapa bagian tertentu tanpa harus menunggu buffering video secara keseluruhan.

Kekurangan:

File hanya dapat dilihat pada saat kita terhubung ke jaringan (online). File akan terhapus setelah selasai dimainkan karena file tidak disimpan di dalam harddisk client.

Untuk melakukan instalasi Streaming Server, yang dibutuhkan adalah:

Komputer sebagai server Komputer sebagai client Hub/switch sebagai penghubung jaringan Kabel jaringan

Beberapa contoh software streaming server:

Adobe Flash Streaming Server Icecast SHOUTcast PlayOn Darwin Streaming Server VideoLAN Client (VLC) Dan lain sebagainya.

24. sebutkan protokol streaming


Real Time Streaming Protocol (RTSP) merupakan protokol jaringan komputer yang dirancang untuk digunakan dalam hiburan dan sistem komunikasi untuk mengendalikan server aliran media (media streaming). Protokol ini digunakan untuk menetapkan dan mengendalikan sesi media antara dua titik ujungnya. Klien dari server media mengeluarkan perintah seperti VCR, seperti playdan pause, untuk mendukung kendali waktu nyatadari berkas media yang dijalankan dari server.

Transmisi aliran data tersebut bukan merupakan tugas protokol RTSP. Sebagian besar server RTSP menggunakan Real-time Transport Protocol (RTP) yang saling melengkapi dengan Real-time Control Protocol (RTCP) untuk pengiriman aliran media, namun beberapa penyedia menerapkan protokol pengiriman dengan hak milik pribadi. Misalnya, server RTSP dari RealNetworks menyediakan protokol miliknya yaitu Real Data Transport.
