18. dimensi tekanan rumus​pelajaran : fisika


Jawaban:

Tekanan = Gaya/Luas (rumus tekanan)

= massa x percepatan /panjang x lebar

= massa x (kecepatan / waktu) /panjang x lebar

= massa x ((jarak / waktu) / waktu) /panjang x lebar

= massa x ((panjang / waktu) / waktu) /panjang x lebar

= [M] x (([L] / [T]) / [T]) / [L] x [L]

= [M] x ([L] / [T]2) / [L]2

= [M] x [T]-2 / [L] , atau

= [M][T]-2[L]-1 (dimensi tekanan)
