18. Phil asked Moe, “What time does the class start?” Change the sentence above into indirect speech! A.Phil asked Moe did the class start on time. B.Phil asked Moe if the class was started. C.Phil asked Moe what time the class started. D.Phil asked Moe what day the class started. E.Phil asked Moe how long the class was. Bantu jawab kak, aku udah banyakin koinnya


Jawaban:

C.

Penjelasan:

Pertama, kalian harus mengubah kalimat tanya “does the class start?” menjadi kalimat pernyataan “the class starts”.

Kedua, jika kalimat pernyataannya dalam bentuk simple present (the class starts), harus diubah ke simple past menjadi “the class started”.

Ketiga, perhatikan sudut pandang pembicara. Dalam indirect speech, kita memakai sudut pandang orang ketiga.

Jadi, jawaban yang tepat adalah C. Phil asked Moe what time the class started.

25. tolong bantu suhu, ntar klo bener gua kasih link nekopoi deh:V​


Jawaban

∠B = 52°

∠C = 78°

Penyelesaian Soal

Ubah perbandingan menjadi,

∠B : ∠C = 2x : 3x

———

A + B + C = 180

50 + 2x + 3x = 180

50 + 5x = 180

5x = 130

x = 26

Maka,

∠B = 2x

= 2(26)

= 52°

∠C = 3x

= 3(26)

= 78°

Panjang sudut keseluruan segitiga = 180 °

∠A = 50

∠B : ∠C = 2 : 3

jadi bisa dibilang kalau

∠B = 2 × ....... atau 2 × x = 2x

∠C = 3 × ..... juga atau 2 × x = 3x

keseluruan segitiga = 180°

180° = ∠A + ∠B + ∠C

180° = 50° + 2x + 3x

180° = 50° + 5x

180° - 50° = 5x

130° = 5x

130/5 = x

26 = x

Yang di cari

B = 2x

= 2 × 26 = 52°

C = 3x

= 3 × 26 = 78°

How to make instant noodle

Ingredients:

a pack of instant noodle.
mustard greens.
an egg.
water. 

Procedure:

First, boil water.
Put instant noodle into the boiling water.
Add seasoning and vegetable oil.
Add a few slices of mustard greens.
Break an egg and put into a bowl.
Beat the egg.
Add the egg into the noodle.
Wait around 3-4 minutes.
Now take your instant noodle and serve in a bowl.
