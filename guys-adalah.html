jumlah Proton = nomor atom

jumlah elektron = jumlah Proton (jika netral)

jumlah elektron = jumlah Proton - muatan

jumlah neutron = nomor massa - nomor atom

39K 19

nomor massa = 39

nomor atom = 19

jumlah Proton = 19

jumlah elektron = 19

jumlah neutron = 39-19=20

40Ca20 2+

nomor massa = 40

nomor atom = 20

muatan = +2

jumlah Proton = 20

jumlah elektron = 20-(+2) = 18

jumlah neutron = 40-20 = 20

79Se34 2-

nomor massa = 79

nomor atom = 34

muatan = -2

jumlah Proton = 34

jumlah elektron = 34-(-2) = 36

jumlah neutron = 79-34 = 45

Jawaban

a). Proton=19; Elektron=19; Neutron=20

b). Proton=10; Elektron=18; Neutron=20

c). Proton=34; Elektron=36; Neutron=45

Pendahuluan

Lambang :

[tex]\boxed{\bf^{A}_ZX}[/tex]

Keterangan :

X = nama unsur

A = nomor massa

Z = nomor atom

∴ Cara menghitung proton, neutron, dan elektron

proton = nomor atom

neutron = nomor massa - nomor atom /proton

elektron = proton (jika muatan netral)

Jika muatan (+), maka jumlah elektron = proton - muatannya

Jika muatan (-), maka jumlah elektron = proton + muatannya

Pembahasan

[tex]a).~^{39}_{19}K[/tex]

Proton = 19

Neutron = 39 - 19 = 20

Karena muatannya netral, maka :

Elektron = proton = 19

[tex]b).~^{40}_{20}Ca^{2+}[/tex]

Proton = 20

Neutron = 40-20 = 20

Karena muatannya +2, maka :

Elektron = proton - 2 = 20-2 = 18

[tex]c).~^{79}_{34}Se^{2-}[/tex]

Proton = 34

Neutron = 79-34 = 45

Karena muatannya -2, maka :

Elektron = proton + 2 = 34+2 = 36
