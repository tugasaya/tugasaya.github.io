Setahu saya, Windows tidak memiliki aplikasi bawaan untuk membuat gambar animasi / GIF

Tapi kamu gunakan saja aplikasi PhotoScape

Langkah-langkah:

A. Membuat GIF

☞ Buka PhotoScape

☞ Buatlah 1 GIF

☞ Pilih beberapa foto sebagai Frame

☞ Atur kecepatan GIF

☞ Tekan "Save"

☞ Selesai

B. Memasang Wallpaper Desktop

☞ Klik kanan di wallpaper desktop

☞ Klik "Properties"

☞ Klik "Desktop"

☞ Cari file GIF tadi dengan klik "Browse"

☞ Pilih file tersebut

☞ Tekan "Apply"

☞ Selelsai

Itu saja yang kutahu, jika ada yang salah.
