20. cari perangakat lunak open sourcedan konvesikan file audio k e format mp3 dan vidio ke format mp4


Perangkat lunak untuk menkonversi format file adalah WinFF
Cara pakainya untuk mengubah MP4 ke MP3 adalah:
- Buka WinFF dan klik Add untuk menambahkan file MP4 yang akan diubah ke MP3
- Lalu, pada kolom Convert to, pilih kategori Audio, dan pada kolom preset pilih MP3
- Lalu, isikan folder untuk menyimpan MP3 hasil konversi dari MP4
- Kemudian klik Convert
- Tunggu sebentar dan file MP4 sudah dikonversi ke MP3
