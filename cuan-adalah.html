18. Ini masalah penyerdehanaan, apakah ada yang tahu cara cepat? atau mungkin tips dan trick utk menyelsaikan masalah seperti ini? terimakasihh banyakk nanti saya doakan dapet cuan banyakk


PEMBAHASAN

• samakan penyebut pada pembilang dan penyebut

a³ - b³ = (a - b)(a² + ab + b²)

a³ + b³ = (a + b)(a² - ab + b²)

pembilang

x²/y + (y² + x²)/(x - y)

= ((x - y)x² + y(y² + x²)) / y(x - y)

= (x³ - x²y + y³ + x²y) / y(x - y)

= (x³ - y³) / y(x - y)

= (x - y)(x² + xy + y²) / y(x - y)

= (x² + xy + y²)/y

penyebut

x³/y² + y²/(x + y) - x⁴/(y³ + xy²)

= (x³(x + y) + y²y² - x⁴) / y²(x + y)

= (x⁴ + x³y + y⁴ - x⁴) / y²(x + y)

= y(x³ + y³) / y²(x + y)

= (x + y)(x² - xy + y²) / y(x + y)

= (x² - xy + y²)/y

soal

= pembilang/penyebut

= ((x² + xy + y²)/y) / ((x² - xy + y²)/y)

= (x² + xy + y²)/(x² - xy + y²)
