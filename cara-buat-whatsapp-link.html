DOCX dan PDF adalah dua format file populer yang banyak digunakan saat ini. DOCX adalah format file milik (proprietary) Microsoft Corporation, sedangkan PDF adalah format file milik (proprietary) Adobe System. DOCX dan PDF saat ini sudah menjadi format file standar terbuka yang dikelola oleh organisasi standar dan dapat digunakan dan diimplementasikan oleh siapapun.

Ini artinya pengembang perangkat lunak aplikasi (software developer) dapat menggunakan dan mengimplementasikan format file tersebut agar aplikasi yang dibuat dapat bekerja baik dengan format file tersebut. Sebagai contoh, aplikasi Microsoft Word secara default menyimpan dokumen dalam format file DOCX, namun juga dapat menyimpan dokumen dalam format PDF.

Baca artikel Mengenal Format File Doc Dan Docx Di Microsoft Word

Dokumen format DOCX banyak dihasilkan kerena kepopuleran aplikasi Microsoft Word, sedangkan PDF banyak digunakan karena keunggulannya. Dukumen PDF tidak bergantung pada platform, baik Windows, macOS atau Linux. Tata letak dan format PDF juga tidak berubah ketika dibuka dengan aplikasi PDF manapun. Dokumen PDF juga sulit untuk diedit sehingga sangat cocok untuk menyimpan dan mendistribusikan dokumen final dan penting.

Apa Itu PDF?

PDF adalah singkatan dari Portable Document File yang diciptakan oleh salah satu Adobe, yaitu John Warnock yang dikembangkan dari tahun 1991 sampai tahun 2000 yang awal mula proses pengambangan ini disebut Proyek Camelot. Pada dasarnya, Warnock ingin siapa pun dapat mengakses file yang dibuat dalam berbagai aplikasi, sehingga mereka dapat dikirim secara elektronik dan diakses melalui komputer apa pun hingga menelurkan PDF yang kita kenal saat ini.

Singkatnya, PDF adalah standar terbuka yang memungkinkan dokumen untuk dipertukarkan dan dibuka oleh siapa saja, terlepas dari sistem operasi, perangkat keras, atau batasan perangkat lunak yang jika tidak akan menyebabkan masalah. Jika Anda pernah menerima dokumen Word dari seseorang yang menggunakan versi perangkat lunak berbeda dari yang Anda miliki di laptop Anda sendiri, mungkin bahkan hanya dari tahun yang berbeda pasti Anda akan kesulitan dalam membuka file tersebut.

Karenanya Organisasi Internasional untuk Standardisasi mempertahankan standar PDF, yang memungkinkan dokumen berisi tautan dan tombol, bidang formulir, audio, dan video dapat dibuka dan diakses. Bahkan menurut Adobe, PDF juga dapat ditandatangani secara elektronik dan siapa pun dapat membaca file di Windows atau Mac dengan menggunakan perangkat lunak Acrobat Reader DC gratis.

24. bagaimana cara menggabungkan beberapa file dengan cara link​


Jawaban:

Untuk menggabungkan beberapa file dengan cara link, Anda dapat mengikuti langkah-langkah berikut:

Pertama, pastikan Anda memiliki semua file yang ingin digabungkan dalam satu folder.

Selanjutnya, buka file yang akan menjadi container atau file yang akan mengandung link-link dari file lainnya.

Pada file container tersebut, tambahkan link-link dari file lain yang ingin digabungkan. Anda dapat menambahkan link dengan menggunakan tag <a> dan menentukan atribut href sesuai dengan lokasi file yang ingin di-link. Contoh: <a href="file1.pdf">File 1</a>

Setelah semua link ditambahkan, simpan file container dan pastikan untuk menyimpan dengan format yang sesuai (misalnya HTML jika file container berupa halaman web).

Buka file container dan pastikan semua link berfungsi dengan benar dan mengarah pada file yang diinginkan.

Selesai dan file-file yang digabungkan sudah bisa digunakan.
