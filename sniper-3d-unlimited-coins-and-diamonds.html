Jawaban:

The original number of $5 coins is 6.The original number of $2 coins is 4.

Penjelasan dengan langkah-langkah:

Number of $5 coins = x

Number of $2 coins = y

Total value is $38:

5x + 2y = 38 ... (1)

If two $5 coins used, then the number of $5 and $2 coins will be the same:

y = x - 2 ... (2)

Substitute the value of y to equation (1):

5x + 2y = 38

5x + 2(x-2) = 38

5x + 2x - 4 = 38

7x - 4 = 38

7x = 38 + 4

7x = 42

x = 42/7

x = 6

y = x-2

y = 6-2

y = 4

Therefore:

The original number of $5 coins is 6.The original number of $2 coins is 4.

14. Perbedaan atau persamaan speed unlimited dan internet unlimited


Perbedaan atau persamaan speed unlimited dan internet unlimited adalah suatu hal yang biasanya menjadi pembeda atau indikator perbedaan pembelian paket kuota atau data internet.

Perbedaan antara Speed Unlimited dengan Internet Unlimited adalah Apabila Speed Unlimited merujuk kepada kecepatan akses internet yang diberikan. Jadi Speed Unlimited adalah istilah yang diberikan jika kecepatan suatu internet tersebut tidak ada batasannya (Unlimited). Namun apabila Internet Unlimited biasanya merujuk ke dalam istilah 'internet sepuasnya' tanpa memperhitungkan kuota data ataupun memori penggunaan, tetapi kecepatan atau speed internet yang diberikan belum tentu unlimited juga.


Persamaan speed unlimited dengan internet unlimited adalah di dalam ketenrangan Unlimited itu sendiri, unlimited dalam bahsa indonesia merujuk ke dalam artian tidak terbatas (tidak diberi pembatasan). Sehingga kedua istilah di atas, sama - sama tidak terbatas namun dalam hal yang berbeda


Pembahasan

Istilah speed unlimited, internet unlimited bisa kita jumapi dengan mudah dalam paket penawaran internet yang dijual oleh ISP (internet service provider) kepada para pelanggan. Selain itu juga bisa kita temua istilah lain seperti, kuota unlimited, FUP, internet speed, dan lain sebagainya.

Jo: x + 362

Linda: 3x

Kelly: x

we can work from that, that means if we add it all together that has a result of:

5x + 362 = 5102

Now, we move the total around,

5102 = 362 + 5x

5102 - 362 = 4740

4740 = 5x

5x = 4740

4740 ÷ 5 = 948

x = 948

Now we know how much Kelly has, now we add it up.

x + 362 = 1310

So,

23. Your questionZaida has P coins and Wan has Q coins. If Zaida gives 20 coins to Wan then both of them have the same number of coins. If Wan gives 50 coins to Zaida, then the ratio of Zaida’s to Wan’s coin is 3:2. Find the number of coin P and coin Q​


Jawab:

Q = 330

P = 370

Penjelasan dengan langkah-langkah:

Zaida has P coins and Wan has Q coins.

If Zaida gives 20 coins to Wan then both of them have the same number of coins.

this can be written as: P - 20 = Q + 20 or P = Q + 40

If Wan gives 50 coins to Zaida, then the ratio of Zaida’s to Wan’s coin is 3:2.

this can be written as: P + 50 : Q - 50 = 3 : 2

P + 50 = 3/2 (Q - 50)

P = 3/2 Q - 125

Q + 40 = 3/2 Q - 125

Q = 165 x 2 = 330

P = 370


24. Helen and Ivan had the same number of coins. Helen had a number of 50-cent coins and 64 20-cent coins. These coins had a mass of 1.134kg. Ivan had a number of 50-cents coins and 104 20-cent coins. A) given that each 50-cent coin is 2.7g more heavier than a 20-cent coin, what is the mass of Ivan's coin in kg?​


Jawab:

1.026 kg

Penjelasan dengan langkah-langkah:

Let n be the number of Helen's 50-cent coins.

It is given that:

Helen and Ivan had the same number of coinsHelen had a number of 50-cent coins and 64 20-cent coinsIvan had a number of 50-cents coins and 104 20-cent coins

104 – 64 = 40

From these facts, we know that Ivan had 40 20-cent coins more than Helen's. Therefore, we can get that the number of Ivan's 50-cent coins which equals to:

n – 40

Let x be the mass of each 50-cent coin, and y be the mass of each 20-cent coin.

At Helen's side, we can get that

nx + 64y = 1.134kg = (1.134 * 1000) g

⇔  nx + 64y = 1,134     .....(i) (in grams)

Given that each 50-cent coin is 2.7g heavier than a 20-cent coin, we can get that

x = y + 2.7      (in grams)

Substitute x to equation (i):

n(y + 2.7) + 64y = 1,134     .....(ii)

Let m be the mass of Ivan's coins, we can get that:

(n – 40)x + 104y = m

⇔  nx – 40x + 104y = m

Then we substitute x with y + 2.7.

⇔  n(y + 2.7) – 40(y + 2.7) + 104y = m

⇔  n(y + 2.7) – 40y – 40(2.7) + 104y = m

⇔  n(y + 2.7) + 64y – 108 = m

Substitute n(y + 2.7) + 64y with 1,134 we got in equation (ii):

⇔  1,134 – 108 = m

⇔  m = 1,026 g = (1,026 / 1,000) kg

⇔ m = 1.026 kg

CONCLUSION:

∴  The mass of Ivan's coins is 1.026 kg.
