13. Sederhanakan bentuk akar berikut.  

a. √112 b. √216 c. √605 d. √800 e. √5.000 f. √0,000121 g. √0,00000324 h. 9√2 + √72 - √578 i. 7√3 + √48 j. 9√5 - √125 + √720

Soal ini terdapat pada buku paket matematika kelas 9 K-2013 revisi 2018 halaman 49.

Menyederhanakn akar bilangan irasional dengan cara  memfaktorkan bilangan kuadrat dengan bilangan biasa.

Sifat Bentuk Akar

[tex]\sqrt{a^{2} } = a[/tex][tex]\sqrt{a} = a^{\frac{1}{2}}[/tex][tex]\sqrt{a} \times \sqrt{b} = \sqrt{ab}[/tex][tex]x \sqrt{a} + y\sqrt{a} = (x + y) \sqrt{a}[/tex][tex]x \sqrt{a} - y\sqrt{a} = (x - y) \sqrt{a}[/tex][tex]\frac{\sqrt{a} }{\sqrt{b}} = \sqrt{\frac{a}{b}}~~ atau~ \sqrt{a} : \sqrt{b} = \sqrt{a : b}~~ , b\neq 0[/tex]

Untuk sifat-sifat dalam menyedehanakan bentuk akar bisa dilihat pada lampiran II

Pembahasan

a. √112

√112 = [tex]\sqrt{16 \times 7}[/tex]

        = √16 × √7

        = 4√7

b. √216

√216 = [tex]\sqrt{36 \times 6}[/tex]

         = √36 × √6

         = 6√6

c. √605

√605 = [tex]\sqrt{121 \times 5}[/tex]

          = √121 × √5

          = 11√5

d. √800

√800 = [tex]\sqrt{400 \times 2}[/tex]

          = √400 × √2

          = 20√2

e. √5.000

√5.000 = [tex]\sqrt{2500 \times 2}[/tex]

             = √2500 × √2

             = 50√2

f. √0,000121

√0,000121 = [tex]\sqrt{121 : 1.000.000}[/tex]

                  = √121 : √1.000.000

                  = 11 : 1.000

                  = 0,011

g. √0,00000324

√0,00000324 = [tex]\sqrt{324 : 100.000.000}[/tex]

                        = √324 : √100.000.000

                        = 18 : 10.000

                        = 0,0018

h. 9√2 + √72 - √578

9√2 + √72 - √578 = 9√2 + [tex]\sqrt{36 \times 2}[/tex] - [tex]\sqrt{289 \times 2}[/tex]

                               = 9√2 + 6√2 - 17√2

                               = 15√2 - 17√2

                               = -2√2

i. 7√3 + √48 - √768

7√3 + √48 - √768

= 7√3 + [tex]\sqrt{16 \times 3}[/tex] - [tex]\sqrt{256 \times 3}[/tex]

= 7√3 + 4√3 - 16√3

= 11√3 - 16√3

= -5√3

j. 9√5 - √125 + √720

9√5 - √125 + √720

= 9√5 - [tex]\sqrt{25 \times 5}[/tex] + [tex]\sqrt{144 \times 5}[/tex]

= 9√5 - 5√5 + 12√5

= 16√5
