f = ___?

frekuensi

f = 7200 / 60

f = 120 Hz ✔️

18. Sebuah gasing awalnya berputar dengan kecepatan sudut 40rad/s , gasing mengalami perlambatan sebesar 0,25 rad/s² karena pengaruh gesekan antara udara dan alastempat berputar gasing. Gasing akan terjatuh setelah?


ω = ω0 + αt
t = (ω - ω0) / α
= (0 rad/s - 40 rad/s) / (-0,25 rad/s²) = 160 s

27. sebuah gasing pada awalnya berputar dengan kecepatan sudut 40 rad.s-1. oleh karena pengaruh gesekan antara udara dan alas tempat berputarnya gasing, gasing mengalami perlambatan sebesar 0,25 rad.s. gasing akan berhenti berputar dalam waktu...beserta caranya..tolong!


Gasing akan berhenti berputar dalam waktu 160 detik

[tex]\texttt{ }[/tex]

PEMBAHASAN

Mari kita ingat kembali beberapa rumus gerak rotasi berikut ini:

[tex]\boxed{\omega_t = \omega_o + \alpha t}[/tex]

[tex]\boxed{\omega_t^2 = \omega_o^2 + 2 \alpha \theta}[/tex]

[tex]\boxed{\theta = \omega_o t + \frac{1}{2} \alpha t^2}[/tex]

dimana:

ωt = kecepatan sudut putaran akhir ( rad/s )

ωo = kecepatan sudut putaran awal ( rad/s )

θ = sudut putaran ( rad )

t = selang waktu

α = percepatan sudut putaran ( rad/s² )

Marilah kita sekarang mencoba menyelesaikan soalnya.

[tex]\texttt{ }[/tex]

Diketahui:

kecepatan sudut putaran awal = ωo = 40 rad/s

percepatan sudut = α = - 0,25 rad/s²

kecepatan sudut putaran akhir = ωt = 0 rad/s

Ditanyakan:

selang waktu = t = ?

Penyelesaian:

[tex]\omega_t = \omega_o + \alpha t[/tex]

[tex]\omega_t - \omega_o = \alpha t[/tex]

[tex]t = (\omega_t - \omega_o) \div \alpha[/tex]

[tex]t = ( 0 - 40 ) \div (-0,25)[/tex]

[tex]t = -40 \div (-0.25)[/tex]

[tex]\boxed{t = 160 \texttt{ detik}}[/tex]
