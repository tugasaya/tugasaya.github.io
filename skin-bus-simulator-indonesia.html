11. jelaskan yang kalian ketahui tentang program simulator simrobotâ€‹


Jawaban:

Istilah simulator robotika dapat merujuk pada beberapa aplikasi simulasi robotika yang berbeda. Misalnya, dalam aplikasi robotika seluler , simulator robotika berbasis perilaku memungkinkan pengguna membuat dunia sederhana dari objek kaku dan sumber cahaya serta memprogram robot untuk berinteraksi dengan dunia ini. Simulasi berbasis perilaku memungkinkan tindakan yang lebih bersifat biologis jika dibandingkan dengan simulator yang lebih biner, atau komputasi. Selain itu, simulator berbasis perilaku dapat "belajar" dari kesalahan dan mampu menunjukkan kualitas antropomorfik dari keuletan.

Salah satu aplikasi paling populer untuk simulator robotika adalah untuk pemodelan 3D dan rendering robot dan lingkungannya. Perangkat lunak robotika jenis ini memiliki simulator yaitu robot virtual, yang mampu meniru gerakan robot sebenarnya dalam amplop kerja nyata. Beberapa simulator robotika menggunakan mesin fisika untuk menghasilkan gerakan robot yang lebih realistis. Penggunaan simulator robotika untuk pengembangan program kontrol robotika sangat disarankan terlepas dari apakah robot sebenarnya tersedia atau tidak. Simulator memungkinkan program robotika untuk dengan mudah ditulis dan di- debug secara off-line dengan versi terakhir dari program yang diuji pada robot yang sebenarnya. Ini terutama berlaku untuk robot industri aplikasi saja, karena keberhasilan pemrograman off-line bergantung pada seberapa mirip lingkungan nyata robot dengan lingkungan simulasi.

Tindakan robot berbasis sensor jauh lebih sulit untuk disimulasikan dan / atau diprogram secara off-line, karena gerakan robot bergantung pada pembacaan sensor seketika di dunia nyata.

=>

Permutasi adalah metode penyusunan kembali angka/objek ke dalam berbagai susunan dimana tidak ada pengulangan yang terjadi. Dalam permutasi ini, urutan menjadi salah satu perhatian, dimana tidak ada anggota yang bertukar tempat.

dengan unsur ganda :

=>

Kombinasi bisa didefinisikan sebagai suatu metode untuk melakukan susunan terhadap benda/objek, dimana terdapat peluang terjadinya pengulangan. Dalam kombinasi, urutan tidak menjadi prioritas, dimana kemungkinan terdapat anggota yang bertukar tempat.

( n, r ) = n! / ( n! - r! )
