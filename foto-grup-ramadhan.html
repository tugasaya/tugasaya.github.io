1.) Jika Kota A Berada Pada Ketinggian 1000 mdpl dengan suhu 20⁰c. Berapakah Suhu kota B dengan ketinggian 700 mdpl?

= 14⁰c.

Cara =

Tinggi A = 1.000 mdpl.

Suhu A = 20⁰c.

Tinggi B = 700 mdpl.

Suhu B = (Tinggi B x Suhu A) ÷ Tinggi A.

Suhu B = (700 mdpl x 20⁰c) ÷ 1.000.

Suhu B = 14.000 ÷ 1.000 = 14⁰C.

2.) Jika Tekanan udara wilayah X 300 mb dan tekanan wilayah Y 500 mb. Kemanakah Angin Akan Bergerak?

= 200 mb.

Cara =

Wilayah X = 300 mb.

Wilayah Y = 500 mb.

Angin Bergerak =

Angin => Wilayah Y - Wilayah X.

Angin => 500 mb - 300 mb.

Angin => 200 mb.
