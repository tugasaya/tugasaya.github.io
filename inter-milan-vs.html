24. sebuah perinter dalam waktu 4 jam dapat mencetak 4.800 lembar tentukan lama perinter bekerja jika perinter tersebut telah mencetak sebanyak 15.600 lembar ​


Penjelasan dengan langkah-langkah:

MATERI - PERBANDINGAN SENILAI

4 jam = 4.800 lembar

x jam = 15.600 lembar, maka:

[tex]\sf \frac{4}{4.800} = \frac{x}{15.600} \\\sf 4 \times 15.600 = 4.800x\\\sf 62.400 = 4.800x\\\sf 4.800x = 62.400\\\sf x = \frac{62.400}{4.800} \\\underline {\boxed {\blue {\sf \bf x = 13 \ jam}}}[/tex]

[tex] \large {\boxed {\blue {\star \:Answered  \: By: \:  \bold {sulkifli2018} \star} } } [/tex]

Diketahui

a1 = 4 jamb1 = 4.800 lembarb2 = 15.600 lembar

Ditanya

a2

JAWAB

Untuk menjawab soal ini, menggunakan prinsip perbandingan senilai :

[tex] \frac{ a1 }{b1} = \frac{a2}{b2} [/tex]

[tex] \boxed{a2 = \frac{a1 \times b2}{b1}} [/tex]

[tex]a2 = \frac{4 \: jam \times 15.600 \: lembar}{4.800 \: lembar} [/tex]

[tex]a2 = \frac{15.600}{1.200} \: jam[/tex]

[tex]a2 = 13 \: jam[/tex]

Kesimpulan

Berdasarkan pembahasan di atas, perinter bekerja selama 13 jam ketika telah mencetak sebanyak 15.600 lembar.
