7. jelaskan cara seorang hacker merusak suatu web.


dengan cara suatu aplikasi dan ilmu pengetahuan yang dimilikinya Salah satunya melalui Password Attacks, yaitu :

Password merupakan sesuatu yang umum jika kita bicara tentang kemanan. Kadang seorang user tidak perduli dengan nomor pin yang mereka miliki, seperti bertransaksi online di warnet, bahkan bertransaksi online dirumah pun sangat berbahaya jika tidak dilengkapi dengan software security seperti SSL dan PGP. Password adalah salah satu prosedur kemanan yang sangat sulit untuk diserang, seorang attacker mungkin saja mempunyai banyak tools (secara teknik maupun dalam kehidupan sosial) hanya untuk membuka sesuatu yang dilindungi oleh password. Ketika seorang attacker berhasil mendapatkan password yang dimiliki oleh seorang user, maka ia akan mempunyai kekuasaan yang sama dengan user tersebut. Melatih karyawan/user agar tetap waspada dalam menjaga passwordnya dari social engineering setidaknya dapat meminimalisir risiko, selain berjaga-jaga dari praktek social enginering organisasi pun harus mewaspadai hal ini dengan cara teknikal. Kebanyakan serangan yang dilakukan terhadap password adalah menebak (guessing), brute force, cracking dan sniffing.

11. bagaimana cara seorang hacker menge hack hp,leptop dan web menggunakan injector


Dengan membuat sebuah jalan pintas layaknya pencuri memasuki rumah yang dinamakan "backdoor". Backdoor adalah script atau kode yang dirancang oleh programmer untuk dijadikan metode agar dapat meng-akses target dan dengan memanfaatkan kelemahan para device (perangkat seperti Smartphone, Laptop, dll) target itu sendiri.

Konsepnya yaitu, backdoor yang berisikan kumpulan kode disimpan menjadi sebuah file, lalu disematkan menjadi satu dalam sebuah aplikasi (software), website, bahkan sebuah gambar maupun video. Setelah itu, ia mengirimkan sebuah umpan ke target, jika berupa software maka, pembuat program ini akan membuat sesuatu agar target memasang program yang dirancang oleh coder. Setelah target meng-install dan menjalankan software tersebut maka akan otomatis mengirimkan sebuah response kepada coder bahwa target sudah masuk ke dalam jaring.

27. Jelaskan yang dimaksud web based email. Sebutkan kekurangan dan kelebihan dari web based email


Web Based Email adalah klien email yang diimplementasikan sebagai aplikasi web dan diakses melalui web browser.


Kelebihan :

Karena beroperasi di halaman situs web, maka fungsi e-mail dapat diakses dari berbagai tempat sepanjang dapat terkoneksi dengan internet.

Tidak memerlukan mail client karena dapat berinteraksi dengan layanan tersebut langsung dari situs web.

Kekurangan :

Pada saat mengakses akun e-mail, koneksi tidak boleh terputus.

E-mail sulit diarsipkan, karena tersimpan di server penyedia layanan e-mail.

Jika server mengalami masalah, ada kemungkinan e-mail dan bahkan akun e-mail dapat hilang begitu saja.
