15. 10. Cara menggunakan fitur Filter, yaitu .... a. Home → Sort & Filter → Filter b. Home → Sort & Filter c. Insert → Sort & Filter d. Insert → Sort & Filter → Filter​


jawab

jwbn yg bnr a. Home → Sort & Filter → Filter

fitur filter dipakai untuk memfilter data dalam suatu range sehingga hanya data yg memenuhi kriteria tertentu yg ditampilkan. ketika menggunakan fitur Filter, pengguna dapat mengatur kriteria filter dengan berbagai cara, seperti memilih nilai tertentu memasukkan teks yg cocok dengan kriteria, atau mengatur rentang nilai. setelah kriteria filter ditentukan, hanya baris data yg memenuhi kriteria dan akan ditampilkan sementara baris data lainny akan disembunyikan. fitur filter sangat berguna ketika bekerja dengan data besar serta kompleks karna memungkinkan pengguna untuk memfokuskan pada data yg relevan dan memudahkan dalam proses analisis data.
