18. Kok www and https? Apa itu 'www' dan 'https'? Jelaskan! Note: #BeraniBertanya


Jawaban: WWW adalah World Wide Web. World Wide Web ini tidak lain adalah situs web atau halaman web yang terhubung ke komputer lokal melalui internet. Situs web ini berisi halaman teks, gambar digital, audio, video, dll.

Hypertext Transfer Protocol Secure adalah ekstensi dari Hypertext Transfer Protocol. Ini digunakan untuk komunikasi aman melalui jaringan komputer, dan banyak digunakan di Internet. Dalam HTTPS, protokol komunikasi dienkripsi menggunakan Transport Layer Security atau, sebelumnya, Secure Sockets Layer.


19. jelaskan istilah dalam internet tersebut.1. web2. www3. website4. web page5. home page6. browser7. download8. upload9.gmail.10. search engine​


Jawaban:

Penjelasan:

6 browser

Penjelasan:

1. Web merupakan halaman situs sistem informasi yang dapat diakses dengan cepat.

2. www (world wide web) adalah kumpulan web server dari seluruh dunia yang berfungsi untuk menyediakan informasi dan data untuk digunakan oleh pengguna internet yang memerlukan.

3. Website adalah kumpulan halaman web yang dapat diakses publik dan saling terkait yang berbagi satu nama domain.

4. Web page adalah suatu halaman yang ditampilkan pada suatu website di internet.

Web page dapat menampilkan tulisan/ teks, gambar, video, dan suara.

5. Homepage adalah halaman muka atau beranda suatu situs web.

6. Browser adalah suatu aplikasi atau program yang dijalankan pada perangkat komputer untuk melihat konten yang ada pada media world wide web (www) dengan memanfaatkan jaringan internet.

7. Download (mengunduh) menerima atau mengambil data atau berkas dari Internet ke komputer kita.

8. Upload (mengunggah) berarti mengirim data atau berkas dari komputer kita ke suatu tempat di Internet.

9. gmail (Google mail) adalah media dari Google untuk mengirim dan menerima pesan.

Pesan yang dikirim bisa dalam banyak bentuk mulai dari dokumen, foto, video serta ragam bentuk dokumen yang lain.

10. Search engine adalah mesin pencari/pelacak/penelusur) yaitu salah satu program komputer yang dirancang khusus untuk membantu seseorang menemukan file-file yang disimpan dalam komputer, misalnya dalam sebuah web server umum di web (www) atau komputer sendiri.

11. VPN (Virtual Private Network) adalah sebuah cara aman untuk mengakses local area network yang berada pada jangkauan tertentu, dengan menggunakan internet atau jaringan umum lainnya untuk melakukan transmisi data paket secara pribadi.
