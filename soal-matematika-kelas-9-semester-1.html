Dengan metode grafik, tentukan penyelesaian dari sistem persamaan berikut untuk x, y € R! Soal: (no 9) y = x + 3 dan 2x + y + 8 = 0 (no 12) 2x - 4y = 6 dan 4x - 2y = -12 Pembahasan

Penyelesaian sistem persamaan dua variabel membutuhkan setidaknya 2 variabel persamaan dalam bilangan bulat.

Metode substitusi: Caranya dengan mengganti persamaan yang satu dan lainnya untuk mendapatkan variabel bernilai bilangan bulat.

Metode eliminasi: Caranya dengan menghilangkan salah satu variabel dengan pengurangan terhadap persamaan tersebut.

Metode grafik: Caranya dengan menentukan titik potong garis terhadap sumbu x dan sumbu y, kemudian digambarkan dalam bentuk grafik terhadap titik potong, sehingga himpunan penyelesaian dapat diketahui jika perpotongan garis terbentuk.

Metode grafik

A.

y = x + 3

2x + y + 8 = 0

B.

2x - 4y = 6

4x - 2y = -12

Menentukan titik potong sumbu x (y = 0) dan titik potong sumbu y (x = 0) untuk masing-masing persamaan.

y = x + 3 ... (1)

Untuk y = 0          Untuk x = 0

0 = x + 3               y = 0 + 3

x = -3                    y = 3

Titik koordinat adalah {x, y} yaitu {-3, 0} dan {0, 3}

2x + y + 8 = 0

Untuk y = 0          Untuk x = 0

2x + 0 + 8 = 0      2(0) + y + 8 = 0

2x = -8                  y = -8

x = -4

Titik koordinat adalah {x, y} yaitu {-4, 0} dan {0, -8}

2x - 4y = 6  ... (3)

Untuk y = 0          Untuk x = 0

2x - 4(0) = 6         2(0) - 4y = 6

2x = 6                   -4y = 6

x = 3                     y = -1,5

Titik koordinat adalah {x, y} yaitu {3, 0} dan {0, -1,5}

4x - 2y = -12  ... (4)

Untuk y = 0          Untuk x = 0

4x - 2(0) = -12       4(0) - 2x = -12

4x = -12                 -2x = -12

x = -3                     x = 6

Titik koordinat adalah {x, y} yaitu {-3, 0} dan {0, 6}

Gambar grafik menunjukkan bahwa perpotongan kedua garis adalah {x, y}, sehingga himpunan penyelesaian adalah sebagai berikut.

a. x = -3,667 dan y = -0,667, ditulis {-3,667, -0,667}

b. x = -5 dan y = -4, ditulis {-5, -4}

Yuukk langsung aja kakak jelasin materi kali ini, oke? Oke! Pertama-tama apa itu fungsi kuadrat? Kakak jawab yaaa! Suatu persamaan yang memiliki variabel dimana variabel tersebut mempunyai pangkat yang paling tinggi dua maka persamaan tersebut disebut sebagai fungsi kuadrat. Bentuk umum dari fungsi kuadrat adalah [tex]y = f(x) = ax^2 + bx + c[/tex], dengan keterangan x disebut sebagai variabel, kemudian a, b adalah koefisien dan c adalah suatu konstanta, serta nilai dari a tidak sama dengan 0. Oke langsung aja yukk disimak penjabaran jawaban soal kali ini.

Tabel atas

Fungsi [tex]f(x) = x^2[/tex]

Sumbu simetri x = 0

Nilai optimum f(0) = 0

Fungsi [tex]f(x) = (x - 1)^2[/tex] artinya dari [tex]y = x^2[/tex] yang digeser 1 langkah ke kanan

Sumbu simetri x = 1

Nilai optimum f(1) = 0

Fungsi [tex]f(x) = (x - 2)^2[/tex] artinya dari [tex]y = x^2[/tex] yang digeser 2 langkah ke kanan

Sumbu simetri x = 2

Nilai optimum f(2) = 0

Fungsi [tex]f(x) = (x + 1)^2[/tex] artinya dari [tex]y = x^2[/tex] yang digeser 1 langkah ke kiri

Sumbu simetri x = -1

Nilai optimum f(-1) = -1

Fungsi [tex]f(x) = (x + 2)^2[/tex] artinya dari [tex]y = x^2[/tex] yang digeser 2 langkah ke kiri

Sumbu simetri x = -2

Nilai optimum f(-2) = 0

Tabel bawah

Fungsi [tex]f(x) = x^2[/tex]

Sumbu simetri x = 0

Nilai optimum f(0) = 0  

Fungsi [tex]f(x) = x^2 + 1[/tex] artinya dari [tex]y = x^2[/tex] yang digeser 1 langkah ke atas

Sumbu simetri x = 0

Nilai optimum f(0) = 1

Fungsi [tex]f(x) = x^2 + 2[/tex] artinya dari [tex]y = x^2[/tex] yang digeser 2 langkah ke atas

Sumbu simetri x = 0

Nilai optimum f(0) = 2

Fungsi [tex]f(x) = x^2 - 1[/tex] artinya dari [tex]y = x^2[/tex] yang digeser 1 langkah ke bawah

Sumbu simetri x = 0

Nilai optimum f(0) = -1

Fungsi [tex]f(x) = x^2 - 2[/tex] artinya dari [tex]y = x^2[/tex] yang digeser 2 langkah ke bawah

Sumbu simetri x = 0

Nilai optimum f(0) = -2

Ayo kita simpulkan

Untuk fungsi [tex]f(x) = (x - s)^2[/tex] diperoleh sumbu simetri x = s dan nilai optimum f(s) = 0 Untuk fungsi [tex]f(x) = x^2 + t[/tex] diperoleh sumbu simetri x = 0 dan nilai optimum f(0) = t Untuk fungsi [tex]f(x) = (x - s)^2 + t[/tex] diperoleh sumbu simetri x = s dan nilai optimum f(s) = t

8 cm
16 cm
64 cm
96 cm
Pembahasan:

Untuk mencari yang ditanya (panjang seluruh rusuk), kita butuh mengetahui panjang rusuknya, bukan panjang diagonal sisinya. Ingatlah rumus Pythagoras karena bentuk sisi kubus yang merupakan persegi dengan sudut yang pastilah siku-siku. Selain memiliki sudut siku-siku, kita juga harus mengingat bahwa segitiga yang terbentuk pada sisi kubus merupakan segitiga siku-siku sama kaki, sehingga untuk mengetahui panjang rusuknya, kita hanya perlu menyingkirkan √2 di belakang bilangan 8. Dengan demikian, panjang rusuk kubusnya adalah 8 cm.

Kemudian, kita dapat mengetahui panjang seluruh rusuk kubus dengan mengalikan panjang rusuk dengan jumlah rusuk kubus (12).

Tinggi badan Doni 160cm dan Andi 180cm.2.Dalam suatu kelas terdapat 7 orang yang menyukai pelajaran matematika adalah perempuan sedangkan 9 orang laki² menyukai pelajaran olahraga maka berapa perbandingan yang menyukai pelajaran matematika dan olahraga.3.Sebuah kolam berukuran panjang 160cm dan lebar 100cm.Berapakah perbandingan panjang dan lebar kolam tersebut?4.Berapakah perbandingan massa Lisa 85kg dan Nisa 50kg.5.Diketahui tinggi badan Maria adalah 160cm dan tinggi badan Alvin 140cm. Tentukanlah perbandingan tinggi badan Maria dan Alvin?BESERTA CARANYA! JAWAB DENGAN BAIK DAN BENAR​


Perbandingan tinggi badan Dono dan Andi adalah 160cm : 180cm = 8 : 9

Perbandingan yang menyukai pelajaran matematika adalah 7 orang perempuan : 0 orang laki-laki = 7 : 0. Perbandingan yang menyukai pelajaran olahraga adalah 0 orang perempuan : 9 orang laki-laki = 0 : 9. Jadi, perbandingan yang menyukai pelajaran matematika dan olahraga adalah 7 : 0 + 0 : 9 = 7 : 9.

Perbandingan panjang dan lebar kolam adalah 160cm : 100cm = 8 : 5.

Perbandingan massa Lisa dan Nisa adalah 85kg : 50kg = 17 : 10.

Perbandingan tinggi badan Maria dan Alvin adalah 160cm : 140cm = 8 : 7.
