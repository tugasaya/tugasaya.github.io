7. Apakah perbedaan antara web browser,web page,web server


Web browser, web page, dan web server adalah tiga istilah yang terkait dengan penjelajahan halaman di dunia maya yang dilakukan oleh pengguna internet.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan perbedaan antara web browser, web page, dan web server. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

WEB BROWSER ADALAH PERANGKAT LUNAK YANG DIPAKAI PENGGUNA INTERNET UNTUK BERSELANCAR DI DUNIA MAYA.

WEB PAGE ADALAH HALAMAN DI INTERNET YANG MENYAJIKAN INFORMASI KEPADA PENGGUNA DAN HANYA DAPAT DIBUKA DENGAN WEB BROWSER.

WEB SERVER ADALAH SISTEM ATAU PERANGKAT YANG DIGUNAKAN PIHAK PENYEDIA KONTEN UNTUK MENYAJIKAN ATAU MENAMPILKAN WEB PAGE DI INTERNET SERTA MENGAKOMODIR AKTIVITAS PENGGUNA DI WEB PAGE TERSEBUT.

17. jelaskan tugas dari web desaigner,web programer,web administrator, web master?


Web desaigner : Mendesain tampilan dari pada website.
Web Programmer : Bertugas dalam melakukan pengcodingan atau pemograman suatu web agar dinamis.
Web administrator : Memaintenance satu buah server, mengerti akan Sistem Operasi Server, baik itu hal instalasi hingga kepada masalah-masalah pada web (troubleshooting)
Web master : Seorang yg sangat mengerti akan kesemua faktor mulai dari desain, program & keamanan server, meskipun demikian seorang webmaster tidak ikut mencampuri pada masing-masing divisi, cukup dgn tugas mempertanggung jawab atas jalannya suatu web (website).

30. siapa penemu film berwarna pertama. kalau contek google,Web,aplikasi nanti aku report dan kalo gak follow aku report​


Soal ✏

• Siapa penemu film berwarna pertama ?

Menjawab ☄

Edward Raymond Turner

Pembahasan

Film ini merupakan film pertama yang menggunakan warna. Pada awal film masih menggunakan teknik warna hitam-putih, lalu saat masuk ke pertengahan saat Dorothy berpetualang ke tempat baru, film langsung berubah warna.

Pathé merupakan perusahaan film yang menjadi pelopor film berwarna. Teknik yang digunakan untuk menghasilkan warna sangatlah kreatif. Mereka mengembangkan penggunaan teknik stencil pada lembar proyeksi. Sejak saat itu, studio film banyak yang terinspirasi untuk membuat film berwarna.

Penggunaan warna tentu menjadi daya tarik tersendiri dalam sebuah film. Mata akan menerima berbagai macam warna yang merangsang ke otak. Terlebih di dalam film The Wizard of Oz, terjadi perubahan warna yang sangat kontras. Untuk membuat teknologi secanggih itu membutuhkan waktu dan kerja keras dari semua pihak yang terkait.

Penjelasan :

☞ Film ☜

Film adalah media komunikasi yang bersifat audio visual untuk menyampaikan suatu pesan kepada sekelompok orang yang berkumpul di suatu tempat tertentu. (Effendy, 1986: 134). Pesan film pada komunikasi massa dapat berbentuk apa saja tergantung dari misi film tersebut. Akan tetapi, umumnya sebuah film dapat mencakup berbagai pesan, baik itu pesan pendidikan, hiburan dan informasi. Pesan dalam film adalah menggunakan mekanisme lambang – lambang yang ada pada pikiran manusia berupa isi pesan, suara, perkataan, percakapan dan sebagainya.
