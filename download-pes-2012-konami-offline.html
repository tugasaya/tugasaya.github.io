8. Pengertian dari : - web browser - Internet address - domain name - hyperlink - url - upload - download - offline - online - search engine


Web Browser: Suatu program/ software yang digunakan untuk menjelajahi internet/ untuk mencari informasi dari suatu web yang tersimpan didalam Komputer

Internet Address: Sebuah alamat pada kompurer agar kompurer bisa saling terhubung dengan komputer lain

Domain Name: Nama unik yang Siberian untuk mengidentifikasi nama server komputer seperti web server/ email server di jaringan komputer/ internet

Hyperlink: Suatu teks yang akan mengadakan ke tampilan dokumen/ halaman lain

URL: Rangkaian karakter menurut format standar tertentu yang digunakan untuk menunjukkan alamat dari suatu Sumber, misalnya: dokumen, file, dan gambar yang terdapat di internet

Upload: Suatu cara untuk mengirimkan file

Download: Sebuah proses dalam pengambilan file- file tertentu yang terdapat di internet, Baik melalui web server, FTP Server, mail server, server, ataupun sistem lain yang identik

Offline: Suatu Istilah untuk debutant saat kita tidak terhubung dengan internet

Online: Istilah saat kita sedang terhubung dengan internet/ dunia maya

Search Engine: Aplikasi yang disediakan Oleh lembaga komersial tertentu yang menyebabkan jasa di bidang internet
