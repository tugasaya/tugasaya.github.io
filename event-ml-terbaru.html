5. Analyze the structure of text! Orientation : Event = Event 1 : Event 2 : Event 3 : Reorientation :


Recount text is a text which presents the writer's experience.

Purpose of this text is to retell the writer's experience.

Generic structure of this text :

1. Orientation

At the beginning of paragraph.

It tells time, place and with whom.

2. Event

It tells the sequence of activities.

Sequence adverbs ( first, second,

next , etc )

3. Reorientation :

At the last paragraph

The conclusion of the writer .



Kesimpulan dari teks recount

Recount text adalah teks yang menceritakan pengalaman si penulis.

Tujuannya adalah untuk menceritakan ulang pengalaman si penulis.

Generic structure :

1. Orientasi > ada di awal paragraf

Menjelaskan waktu , tempat kejadian


2. Event > kejadian berurutan . Bisa diidentifikasi keterangan nya ( kata first, second, next , dll )

3. Reorientasi > kesimpulan dari penulis tentang pengalaman nya. Biasanya ada di bagian penutup paragraf.
