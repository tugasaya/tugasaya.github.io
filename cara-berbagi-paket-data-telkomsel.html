11. nusa membeli paket data 8GB 30 hari .setelah 10 hari sudah terpakai 40% sisia paket data nusa adalah ... byte pake cara​


paket data = 8G = 8.000.000 byte

terpakai = 40%

sisa paket data

= paket data × (100% - terpakai)

= 8.000.000 × (100% - 40%)

= 8.000.000 × 60%

= 4.800.000 byte
