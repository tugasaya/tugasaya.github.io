1). 1 liter = .... dm³

• 1 liter = 1 dm³

• 1 ml = 1 cm³

• 1 ml = 1 cc

• 1 cm³ = 1 cc

Jadi,

1). Hasil dari 1 liter jika diubah menjadi satuan volume dm³ adalah 1 dm³.

18. apa perbedaan static link library, dynamic link library, dan import library?​


Jawaban:

Static library(juga dikenal sebagai archive) terdiri dari rutinitas yang disusun dan dihubungkan langsung ke program anda. Ketika anda mengkompilasi sebuah program yang menggunakan static library, semua fungsi dari static library menjadi bagian dari eksekusi. Pada Windows, static library biasanya memiliki ekstensi .lib, Sedangkan di linux, perpustakaan statis biasanya memiliki ekstensi .a (archive). Satu keuntungan dari static library adalah bahwa anda hanya harus mendistribusikan executable agar pengguna untuk menjalankan program anda. Karena library menjadi bagian dari program anda, ini memastikan bahwa versi yang tepat dari library selalu digunakan dengan program anda. Juga, karena static library menjadi bagian dari program anda, anda dapat menggunakannya sama seperti fungsi yang sudah anda tulis untuk program anda sendiri. Pada sisi negatifnya, karena salinan library menjadi bagian dari setiap executable, maka ini dapat menyebabkan banyak ruang terbuang. Static library juga tidak dapat diperbarui dengan mudah, untuk memperbarui library, seluruh executable perlu diganti.

Dynamic library (juga disebut shared library) terdiri dari rutinitas yang dimuat ke dalam aplikasi anda pada waktu dijalankan. Ketika anda mengkompilasi sebuah program yang menggunakan dynamic library, library tidak menjadi bagian dari eksekusi, tetap sebagai unit terpisah. Pada Windows, dynamic library biasanya memiliki ekstensi .dll (dynamic link library), Sedangkan di Linux, dynamic library biasanya memiliki ekstensi .so (shared object). Salah satu keuntungan dari dynamic library adalah bahwa banyak program dapat berbagi satu salinan, yang menghemat ruang. Mungkin keuntungan yang lebih besar adalah bahwa dynamic library dapat diperbarui ke versi yang lebih baru tanpa mengganti semua executable yang menggunakannya.

28. Dimana letak perbedaan antara link standard dan link email serta penulisan link standard dan link email yang benar


Perbedaannya yaitu link standart ketika di klik akan membuka laman yg ditujukan , dan link email akan membuka peramban software email (Biasanya MS.Outlook) untuk mengirim email ke email yg ditujukan .

Penulisan Link standart:<a href="(laman)">(teks)</a> .contoh hyperlink : <a href="brainly.com">klik disini</a> , pada laman akan ada teks klik disini yg berwarna biru ,ketika di klik akan menuju laman brainly.com

Penulisan link email : <a href = "mailto:(email kamu)">(kalimat kamu)</a> , misal <a href = "mailto:syafrialdiyogi@gmail.com">Klik Link untuk mengirim</a>
