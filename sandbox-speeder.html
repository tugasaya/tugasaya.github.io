Jawaban

I like visiting Dolane park because there are many fun things to do and play. There is a big chute to slide on and a huge sandbox to play in. There are some swings. There is a small pond under the tree. There are some colorful fish in it. Pets are not allowed in the park, so there aren't pets to distract us. There is also a lot of space for us to run around. It sometimes gets hot because there aren't many trees to give us shade, but there is a fountain where we can drink water. It's the best place in the town for children. There isn't any other place as fun as the park.

Terjemahan:

Aku suka mengunjungi taman Dolane karena ada banyak hal menyenangkan untuk dilakukan dan dimainkan. Ada perosotan besar untuk meluncur dan bak pasir besar untuk dimainkan. Ada beberapa ayunan. Ada sebuah kolam kecil di bawah pohon. Ada beberapa ikan berwarna-warni di dalamnya. Hewan peliharaan tidak diperbolehkan di taman, jadi tidak ada hewan peliharaan yang mengganggu kita. Ada juga banyak ruang bagi kita untuk berlarian. Kadang-kadang suasana bisa menjadi panas karena tidak ada banyak pohon yang memberi kita naungan, tetapi ada sebuah air mancur di mana kita bisa minum air. Ini adalah tempat terbaik di kota untuk anak-anak. Tidak ada tempat lain yang menyenangkan selain taman ini.

Penjelasan

There is (ada) dan there isn't (tidak ada) digunakan pada kata benda tunggal.There are (ada) dan there aren't (tidak ada) digunakan pada kata benda jamak.

Pembahasan Jawaban

1. There are many fun things to do and play.

There are ➔ Many fun things (ada banyak hal menyenangkan, jamak)

2. There is a big chute to slide on.

There is ➔ A big chute (ada sebuah perosotan, tunggal)

3. There are some swings.

There are ➔ Some swings (ada beberapa ayunan, jamak)

4. There is a small pond under the tree.

There is ➔ A small pond (ada sebuah kolam kecil, tunggal)

5. There are some colorful fish in it.

There are ➔ Some colorful fish (ada beberapa ikan warna warni, jamak)

6. There aren't pets to distract us.

There aren't ➔ Pets (tidak ada hewan peliharaan, jamak)

7. There is also a lot of space for us to run around.

There is ➔ A lot of space (ada banyak ruang, tunggal)Kata space (ruang) di sini merupakan uncountable noun (tidak bisa dihitung). Uncountable noun selalu diikuti bentuk kata kerja singular.

8. There aren't many trees to give us shade.

There aren't ➔ Many trees (tidak ada banyak pohon, jamak)

9. There is a fountain where we can drink water.

There is ➔ a fountain (ada sebuah air mancur, tunggal)

10. There isn't any other place as fun as the park.

There isn't ➔ any other place (tidak ada tempat lain, tunggal)
