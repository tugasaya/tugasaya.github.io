10. jika rangkaiam ac tetdapat 2H dan kecepatan listrik 30rod/s hitung nilai xl


Materi : Listrik
Sub materi : Listrik AC
Mapel : Fisika
Kelas : XII SMA (KTSP)
Keyword : Listrik Bolak-Balik, Reaktansi Induktif

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
L = 2 H
ω = 30 rad/s

Ditanya :
xL = ?

Penyelesaian :
Reaktansi induktif merupakan hambatan yg muncul pada induktor ketika dialiri arus AC

Analisis soal :
Suatu induktor L dialiri arus AC yg memiliki kecepatan sudut ω. Ditanyakan berapakah reaktansi induktif nya.

Perhitungan :
xL = ωL
xL = 30 . 2
xL = 60 ohm

Jadi, reaktansi induktif rangkaian ac tersebut adalah 60 ohm

19. Nilai x yang memenuhi persamaan l2-xl²-l2-xl=3l2-xl+5 adalah​


Penjelasan dengan langkah-langkah:

|2 - x|² - |2 - x| = 3|2 - x| + 5

misal |2 - x| = p, maka :

p² - p = 3p + 5

p² - p - 3p - 5 = 0

p² - 4p - 5 = 0

(p + 1)(p - 5) = 0

p = -1 atau p = 5

• untuk p = -1

|2 - x| = -1 (tidak memenuhi, karena nilai mutlak tidak mungkin bernilai negatif)

• untuk p = 5

|2 - x| = 5

2 - x = -5 atau 2 - x = 5

-x = -5 - 2 atau -x = 5 - 2

-x = -7 atau -x = 3

x = 7 atau x = -3

Jadi, HP = {-3, 7}
