16. berapa hasil pecahan berpangkat 2per6 pangkat-3


[tex] { \frac{2}{6} }^{ - 3} = 27[/tex]

Pembahasan :

Bilangan berpangkat adalah bilangan yang berbentuk

[tex] {a}^{b} [/tex]

Maksudnya a dikalikan a sebanyak b kali. Contoh :

[tex] {2}^{4} = 2 \times 2 \times 2 \times 2 = 16[/tex]

Rumus menghitung bilangan berpangkat :

[tex] \boxed{ { \frac{a}{b} }^{ - c} = { \frac{b}{a} }^{c} } [/tex]

[tex] \boxed{ {a}^{ - b} = \frac{1}{ {a}^{b} } }[/tex]

Penyelesaian soal di atas :

[tex] { \frac{2}{6} }^{ - 3} [/tex]

[tex] = { \frac{6}{2} }^{3} [/tex]

[tex] = {3}^{3} [/tex]

[tex] = 27[/tex]

Detail jawaban :

Kelas : 9

Mapel : Matematika

Bab : Bilangan berpangkat

Kode : 9.2.1

Kata kunci : Bilangan berpangkat, rumus, menghitung bilangan berpangkat

Pertanyaan

[tex] = \frac{2}{6} ^{ - 3} [/tex]

Menggunakan

[tex] = \frac{1}{a} ^{ - n} = {a}^{n} [/tex]

Maka

[tex] = \frac{\sf{ \cancel{2 }}}{\sf{ \cancel{6 }}} ^{ - 3} \\ \\ = \frac{1}{3} ^{ - 3} \\ \\ = {3}^{3} \\ \\ = \boxed{ \: 27 \: }[/tex]

28. pangkat pecahan dari 125 pangkat -2/3 adalah


125 = 5^3
125^-2/3 = 5^-2 = 0.04
125^(-2/3) = 1/(125^(2/3))
                 = 1/(5^(3)(2/3))
                 = 1/5^2
                 = 1/25

Jadi, 125^(-2/3) = 1/25
