13. Berdasarkan format Nomor Induk Kependudukan (NIK) di atas, jika diketahui 3 orang lahir di tempat dan tahun yang sama, berapa peluang ketiganya memiliki NIK yang berbeda?​


Materi : Peluang

Diketahui

- 3 bayi lahir dengan data yaitu

=> Tempat yang sama lahirnya

• Kode Provinsi : sama

• Kode Kabupaten/Kota : sama

• Kode Kecamatan : sama

=> Lahir pada tahun yang sama

• Tanggal : tidak sama [ 1 - ~ ] { tergantung bulan } [ _ ]

• Bulan : tidak sama [ 1 - 12 ] [ _ ]

• Tahun : sama

Kode Komputerisasi : tidak sama [ _ ]

Ditanya

Peluang 3 bayi memiliki hasil NIK yang berbeda

Dijawab

Yang Membedakan

Tanggal [ 01 - ~ ]

Bulan [ 01 - 12 ]

Tahun [ Kabisat / tidak ? ]

Kode Komputerisasi [ 0-9 0-9 0-9 0-9 ]

Kalau tidak tahun kabisat

Tahun = 365 hari => 365 kemungkinan

Tahun = 12 bulan => 12 kemungkinan

=> 4.380 kemungkinan

Kalau tahun kabisat

Tahun = 366 hari/Tanggal => 366 kemungkinan

Tahun = 12 bulan => 12 kemungkinan

=> 4.392 kemungkinan

---

Kode Komputerisasi [ setiap digit ada kemungkinan salah satu dari 10 angka ] => 10⁴ = 10.000 kemungkinan

Maka untuk tidak sama adalah

kabisat => ( 1/43.920.000 )³ = 1/84.720.204.288.000.000.000.000 untuk 3 bayi sama, maka untuk tidak sama [ 99,97716% ]

tidak kabisat => ( 1/43.800.000 )³ = 1/84.027.672.000.000.000.000.000 untuk 3 bayi sama, maka untuk tidak sama [ 99,97723% ]
