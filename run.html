12. Change into negative the children run home. a.the children run not home. b.the children doesn't run home. c.the children did not run home. d.the children did not runned home.


Jawaban:

C. the children did not run home

karna itu past tense dan V2 dari run itu tetap run

jadinya

the children didn't run home

Jawaban:

Present tense

Penjelasan:

(+) The children run home

(-) The children don't run home

(?) Do the children run home ?

Jawaban yang tepat: ⤵️

The children don't run home

Alasan :

1. Noun merupakan bentuk plural dari child.

2. Noun yang bersifat plural atau jamak tidak mengalami penambahan s/es pada kata kerja (verb)

3. Verb pada awal kalimat merupakan verb 1. Yaitu RUN

4. Run merupakan irregular verb ( Run, Ran, Run)

5. Kalimat tersebut tidak termasuk pada past Tense sebab, pada awal soal tidak terjadi perubahan verb ke bentuk ke-2.

Jawaban a tidak tepat karena not harus diiringi dengan kata bantu . seperti do, will etc

Jawaban b kurang tepat karena children adalah bentuk jamak dan Dan jamak menggunakan DO bukan DOES

Jawaban c dan d tidak tepat karena pada soal tidak terjadi perubahan kata kerja ke bentuk ke - 2. Sehingga kalimatnya termasuk present tense
