Ibu Tutis : 8 hari sekali

Ibu Rethna : 9 hari sekali

Hari ini mereka berdua bertemu dipasar, hari keberapa mereka akan bertemu lagi?

Jawab :

Gunakan Kpk

8 = 2 x 2 x 2 / 2³

9 = 3 x 3 / 3²

KPK = 2 x 2 x 2 x 3 x 3

KPK = 4 x 6 x 3

KPK = 24 x 3

KPK = 72 hari

Jadi, hari ke-72 mereka akan bertemu lagi.
