11. Suatu pekerjaan dapat diselesaikan oleh 8 orang laki2 dan 12 orang perempuan dalam waktu 10 hari. Sedangkan jika dikerjakan oleh 6 orng laki2 dan 8 orang perempuan pekerjaan itu slesai dalam waktu 14 hari. Berapa waktu yang diperlukan untuk menyelesaikan pekerjaan itu jika dikerjakan oleh: a. Seorang laki2 saja? b. Seorang perempuan saja?


misal seorang laki-laki dapat mengerjakan dalam x jam, dan wanita y jam

1/x + 1/x + 1/x + .... + 1/x + 1/y + 1/y + .... + 1/y = 1/10
8/x + 12/y = 1/10
(8y + 12x) / xy = 1/10
xy = 10(8y + 12x)
xy = 80y + 120x

1/x + 1/x + 1/x + .... + 1/x + 1/y + 1/y + ... + 1/y = 1/14
6/x + 8/y = 1/14
(6y + 8x) / xy = 1/14
xy = 14(6y + 8x)
xy = 84y + 112x

xy = xy

80y + 120x = 84y + 112x
84y - 80y = 120x - 112x
4y = 8x
y = 8/4 x
y = 2x

xy = 80y + 120x
x(2x) = 80(2x) + 120x
2x² = 160x + 120x
2x² = 280x
x² = 280/2 x
x² = 140x
x = 140 hari

y = 2x
y = 2(140)
y = 280 hari

Jadi, seorang laki-laki dapat mengerjakannya selama 140 hari, dan seorang perempuan dapat mengerjakannya selama 280 hari.

= 25 - 19
