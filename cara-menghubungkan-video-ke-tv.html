13. cara ngatasi format video yg tidak support di tv samsung t4003? file video nya harus format apa? ​


Jika TV Samsung T4003 tidak dapat memutar file video tertentu karena format yang tidak didukung, Anda dapat mengubah format file video tersebut menjadi format yang didukung oleh TV tersebut. Format video yang didukung oleh TV Samsung T4003 antara lain:

AVI (DivX, XviD)

MP4 (MPEG4, H.264)

MKV (H.264, MPEG4, VP8)

WMV (VC1)

Untuk mengubah format file video, Anda dapat menggunakan perangkat lunak konversi video, seperti HandBrake, Freemake Video Converter, atau Any Video Converter. Berikut ini adalah langkah-langkah umum untuk mengkonversi format file video menggunakan HandBrake:

Unduh dan instal HandBrake dari situs web resminya.Buka HandBrake dan pilih file video yang ingin Anda konversi.Pilih format keluaran yang diinginkan, seperti MP4 atau MKV, pada menuKonfigurasi pengaturan keluaran, seperti resolusi, bit rate, dan codec, jika diperlukan.Klik tombol "Start" untuk memulai konversi. Proses konversi dapat memakan waktu tergantung pada ukuran file video yang ingin Anda konversi dan kecepatan komputer Anda.Setelah konversi selesai, salin file video yang baru dibuat ke perangkat penyimpanan eksternal yang dapat terhubung ke TV Samsung T4003, seperti flash drive atau hard drive eksternal.Sambungkan perangkat penyimpanan eksternal ke TV Samsung T4003 dan pilih file video yang ingin Anda putar.

Dengan mengubah format file video menjadi format yang didukung oleh TV Samsung T4003, Anda seharusnya dapat memutar video tanpa masalah. Pastikan juga untuk memperhatikan resolusi video dan bit rate yang Anda pilih saat mengkonversi, karena kualitas video dapat berdampak pada pengalaman menonton.

21. video tutorial tentang cara menghubungkan 2 router, 2.Switch, 4 PC,​


Jawaban:

Cara menyambungkan 2 router,2 switch dan 4 pc di cisco packet tracer sebagai berikut:

1.Klik Router0

2.Klik Router1

3.Sambungkan Router0 dengan Router1 menggunakan kabel crossover atau bisa menggunakan otomatis.Router0 sebagai FastEthernet 0/0,dan Router1 Juga sebagai FastEthernet 0/0,

Setelah membuat sesuai gambar diatas kita melanjutkan mengatur Router0 dan juga Router1 yaitu sebagai berikut:

1.Klik Router0

2.Klik Config

3.Klik FastEthernet 0/0

4.Centang Port Status Menjadi ON

5.Isi IP Address dengan 192.168.1.1,maka saat di klik ke subnet mask akan keluar 255.255.255.0

Setelah membuat sesuai gambar diatas kita melanjutkan mengatur Router1 yaitu sebagai berikut:

1.Klik Router1

2.Klik Config

3.Klik FastEthernet 0/0

4.Centang Port Status Menjadi ON

5.Isi IP Address dengan 192.168.1.2,maka saat di klik ke subnet mask akan keluar 255.255.255.0

setelah sesuai gambar diatas selanjutnya menambahkan switch di sebelah kiri Router0 dan sebelah kanan Router1 dengan cara seperti ini:

1.Klik switch0

2.Sambungkan switch0 dengan Router0 menggunakan kabel straight(router0 sebagai FastEthernet 0/1 )

3.Klik switch1

4.Sambungkan switch1 dengan Router1 menggunakan kabel straight(router1 sebagai FastEthernet 0/1 )

setelah sesuai gambar diatas selanjutnya menambahkan pc di sebelah kiri switch0 dan sebelah kanan switch1 dengan cara seperti ini:

1.Klik PC0

2.Sambungkan PC0 dengan switch0 menggunakan kabel straight

3.Klik PC1

4.Sambungkan PC1 dengan switch0 menggunakan kabel straight

5.Klik PC2

6.Sambungkan PC2 dengan switch1 menggunakan kabel straight

7.Klik PC3

8.Sambungkan PC3 dengan switch1 menggunakan kabel straight

setelah sesuai gambar diatas selanjutnya kita mengatur Router0 untuk switch0,dengan cara seperti dibawah ini:

1.Klik Router0

2.Klik Config

3.Klik FastEthernet 0/1

4.Centang Port Status Menjadi ON

5.Isi IP Address dengan 192.168.2.1,maka saat di klik ke subnet mask akan keluar 255.255.255.0

Setelah sudah sesuai diatas,selanjutnya mensetting RIP

6.Klik Rip

7.Isi Network 192.168.1.1 di add,lalu isi Network lagi 192.168.2.1 lalu di add(ip yang digunakan sesuai IP di FastEthernet 0/0 dan FastEthernet 0/1)

setelah sesuai gambar diatas,selanjutnya mensetting IP address PC0 dengan settingan seperti di bawah ini:

IP Address:192.168.2.2

Subnet Mask:255.255.255.0

Default Gateway:192.168.2.1(sesuai ip address di FastEthernet 0/1 di router0)

selanjutnya mensetting IP address PC1 dengan settingan seperti di bawah ini:

IP Address:192.168.2.3

Subnet Mask:255.255.255.0

Default Gateway:192.168.2.1(sesuai ip address di FastEthernet 0/1 di router0)

setelah selesai mengatur Router0 dan switch0,selanjutnya kita mengatur Router1 dan switch1 dengan cara seperti ini:

1.Klik Router1

2.Klik Config

3.Klik FastEthernet 0/1

4.Centang Port Status Menjadi ON

5.Isi IP Address dengan 192.168.3.1,maka saat di klik ke subnet mask akan keluar 255.255.255.0

Setelah sudah sesuai diatas,selanjutnya mensetting RIP

6.Klik Rip

7.Isi Network 192.168.1.2 di add,lalu isi Network lagi 192.168.3.1 lalu di add(ip yang digunakan sesuai IP di FastEthernet 0/0 dan FastEthernet 0/1)

setelah sesuai gambar diatas,selanjutnya mensetting IP address PC2 dengan settingan seperti di bawah ini:

IP Address:192.168.3.2

Subnet Mask:255.255.255.0

Default Gateway:192.168.3.1(sesuai ip address di FastEthernet 0/1 di router0)

selanjutnya mensetting IP address PC3 dengan settingan seperti di bawah ini:

IP Address:192.168.3.3

Subnet Mask:255.255.255.0

Default Gateway:192.168.3.1(sesuai ip address di FastEthernet 0/1 di router0)

setelah selesai melakukan semua itu,kita hanya tinggal melakukan Ping,lakukan hingga success

Lantas mengapa saya membuat artikel yang seperti ini? Tutorial membuat video slideshow ini Insya Allah nantinya akan berlanjut ke tutorial yang lebih berguna lagi buat anda, karena saya akhir-akhir ini sedang melakukan uji coba pembuatan video dengan beberapa software, yang pastinya cari software yang gratisan untuk kemudian diupload ke Youtube.

Pembuatan video slideshow yang dikumpulkan dari beberapa foto pribadi ini sangat berguna sekali. Sebab dengan hanya membuat satu video saja maka anda sama dengan memiliki koleksi album foto kenangan. Tidak perlu dilihat satu-satu foto-foto anda, cukup diputar sekali saja maka seluruh foto yang sudah anda jadikan video akan tertampil semuanya.

Langsung saja ya tidak usah basa basi, berikut tutoral cara membuat video slideshow dari gambar atau foto.

1. Pastikan anda sudah memiliki atau mengumpulkan foto/gambar yang mau dijadikan video slideshow. Atau anda bisa juga cari gambar di Google dan simpan gambar-gambar tersebut. Dan juga siapkan satu lagu untuk disisipkan dalam video nantinya

2. Download software gratisan Total Video Converter

3. Instal Total Video Converter pada komputer anda. Saya anggap anda-anda sekalian sudah jago urusan instal menginstal ya

4. Jalankan program Total Video Converter tadi

5. Klik pada tanda panah di samping New Task kemudian pilih Import Photos (Photos->Video)

￼

6. Masukkan semua gambar/foto yang mau dijadikan video kemudian klik Add Pictures, atau bisa juga sebelumnya foto anda masukkan dalam satu folder, pilih folder kemudian klik Add Folder

￼

7. Langkah selanjutnya tambahkan satu file musik terserah anda, kemudian klik Add Audio

￼

8. Bagian bawah menu klik Settings, tentukan pengaturan video anda, misal masing-masing foto ditampilkan selama 4 detik dan efek ditampilkan 2 detik, kemudian atur resolusi video atau biarkan default 720×480, klik OK

￼

￼

9. Pengaturan video selesai kemudian klik OK

￼

10. Langkah selanjutnya atur file ekstensi video yang anda inginkan, misal video mau anda convert ke FLV

￼

11. Klik Convert Now, maka proses convert akan berlangsung, tunggu hingga proses selesai

￼

￼

12. Jika proses konversi sudah selesai maka anda akan dibawa ke folder file hasil konversi video tadi, tinggal putar video slideshow anda. Jika dirasa ada yang kurang silahkan atur-atur sendiri mulai dari awal.
