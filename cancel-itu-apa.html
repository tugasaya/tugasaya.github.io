30. if it.....(rain), we.......(cancel) the match​


Dilihat dari struktur kalimat nya, kalimat di atas merupakan conditional sentence. Karena tidak ada perintah mau menggunakan Type 1, Type 2 atau Type 3, Maka disini saya akan menjawab ketiga type tersebut.

PEMBAHASAN
==============
RUMUS
1). Conditional Type 1
If + simple present, conditional (=will + V1)

2). Conditional Type 2
If + simple past, conditional (=would + V1)

3). Conditional Type 3
If + past perfect, conditional (=would + have + V3)

Jawaban :
Type 1 => If it RAINS, we WILL CANCEL the match.

Type 2 => If it RAINED, we WOULD CANCEL the match.

Type 3 => If it HAD RAINED, we WOULD HAVE CANCELED the match.
