10. simpangan kuartil dari data 46,56,59,64,65,66,72,74,75,82 adalah..


Jawab:

Simpangan kuartil dari data tersebut adalah 7 ¹/₂

Penjelasan dengan langkah-langkah:

Diketahui: 46, 56, 59, 64, 65, 66, 72, 74, 75, 82

Ditanya: simpangan kuartilnya

Jawab:

Simpangan kuartil adalah jangkauan antarkuartil. Simpangan kuartil merupakan setengah dari selisih antara kuartil atas (Q3) dengan kuartil bawahnya (Q1). Sebelum menghitung kuartil, kita harus mengurutkan datanya terlebih dahulu, dari data yang nilainya terkecil ke yang terbesar.

Simpangan kuartil = ¹/₂ ( Q3 - Q1 )

Letak kuartil

Untuk N data genap, letak kuartil Qi = i ( n + 1 ) / 4Untuk N data ganjil, letak kuartil:

        Q₁ = data ke ( n + 2 ) / 4

        Q₃ = data ke ( 3n + 2 ) / 4


Kembali ke soal, diketahui datanya: 46, 56, 59, 64, 65, 66, 72, 74, 75, 82

N = 10, data genap

Q1 = data ke ( 10 + 2 ) / 4

    = data ke 3

Q1 = 59


Q3 = data ke ( 3 . 10 + 2 ) / 4

     = data ke 8

Q3 = 74

Jadi, simpangan kuartilnya adalah:  

Simpangan kuartil =  ¹/₂ ( Q3 - Q1 )

                               = ¹/₂ ( 74 - 59 )

                               = ¹/₂ . 15

                               = 7 ¹/₂

Jadi, nilai simpangan kuartilnya adalah 7 ¹/₂
