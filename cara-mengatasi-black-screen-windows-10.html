10. Cara mengatasi thumbnail file hilang di windows 10 gimana?​


Jawaban:

CARA 1

Sebelum menggunakan cara yang lebih kompleks, ada baiknya kita mengecek terlebih dahulu apakah dukungan  gambar mini diaktifkan oleh File Explorer. Namun, terkadang tiap folder memiliki pengaturan tersendiri sehingga jika Anda mengatur secara global, pada folder tertentu tetap tidak menampilkan thumbnail.

Untuk mengaktifkan gambar cuplikan, klik tab View lalu klik salah satu pilihan di bawah ini:

Extra large icons Large icons Medium icons Tiles Content[ LIHAT GAMBAR 1 ]

Semua mode tampilan di atas mendukung gambar cuplikan. Pilih salah satu diantara pilihan di atas untuk mengaktifkan gambar mini.

CARA 2

Meskipun sudah menggunakan mode tampilan yang mendukung gambar mini, thumbnail sering kali tetap tidak diaktifkan karena File Explorer dikonfigurasi agar hanya menampilkan icon saja tidak dengan cuplikan. Ketika pengaturan tersebut tidak diaktifkan, File Explorer hanya menampilkan icon dari aplikasi yang digunakan untuk membuka file yang berkaitan.

Berikut adalah cara mengaktifkan thumbnail di Windows 10

Langkah 1: Buka File Explorer. Klik menu File lalu klik Change folder and search options untuk membuka pengaturan folder. Ketika ada file yang sedang terpilih, Windows 10 hanya akan menampilkan Options saja bukan “Change folder and search options”. Klik menu tersebut juga akan membuka jendela yang sama.

[ LIHAT GAMBAR 2 ]

Langkah 2: Pindah ke tab View. Dibagian Advanced settings, hilangkan centang pada opsi dengan judul Always show icons, Never thumbnails, dan klik tombol Apply untuk menerapkan perubahan.

[ LIHAT GAMBAR 3 ]

CARA 3

Jika tampilan gambar mini terlihat tidak jelas (blur), kemungkinan besar hal tersebut dikarenakan file cache thumbnail yang rusak. Anda dapat membersihkan dan mereset cache thumbnail melalui Disk Cleanup dengan mencentang pilihan Thumbnails.

CARA 4

Mungkin Anda sudah mengetahui, File Explorer di Windows 10 tidak dapat menampilkan gambar mini untuk semua jenis file. Meskipun sebagian besar tipe file gambar / video sudah di dukung, format seperti .mkv dan .flv sering kali ditampilkan hanya icon-nya saja.

Jika Anda menginginkan gambar mini untuk semua jenis file gambar dan video, maka Anda harus menginstall thumbnail generator yang dijuluki Icaros. Aplikasi mungil tersebut tersedia secara gratis dan mampu membuat thumbnail hingga 15 jenis file.

Penjelasan:

Sumber : https://www.aplikasipc.com/mengatasi-gambar-mini-thumbnail-tidak-muncul-di-file-explorer/

14. cara mengatasi ram usage di laptop windows 10 jangan asal ​


Jawaban:

Penyebab RAM Usage Tinggi

Sebelum kamu tahu cara untuk mengatasinya, kamu harus tahu dulu penyebab yang mengakibatkan pemakaian RAM tinggi.

Beberapa penyebabnya:

Kapasitas RAM yang memang kecil.

Pemakaian aplikasi secara bersamaan dalam satu waktu.

Terinfeksi virus

Menggunakan background atau tema dengan size besar.

Menggunakan beberapa efek dan layanan sistem operasi.

Berikut langkah-langkahnya:

Masuk ke tab Startup

Kemudian pilih program mana saja yang tidak terlalu penting

Ubah status program yang “Enabled” menjadi “Disabled”.

17. cara mengatasi laptop yang black screen gimana ya? please bantu ya.


1Masuk ke Safe Mode

Solusi awal yang bisa dilakukan sebagai cara mendapatkan Windows yang baik kembali adalah dengan masuk ke mode aman sistem  atau safe mode windows 7. Langkah untuk masuk ke Safe mode Windows 7 sendiri adalah sbb :

Nyalakan laptop;

Sebelum masuk tampilan logo Windows 7, tekan tombol F8, beberapa kali, hingga masuk menu Boot Option Windows 7;

Dari opsi yang ada, pilih Safe Mode;

Setelah tampil safe mode, tunggu sekitar 5-10 menit lalu restart laptop dan nyalakn kembali laptop, langkah ini pernah diuji dan memang jika masalahnya tidak parah bisa menjadikan laptop menyala normal kembali;

Biasanya masalah seperti ini diakibatkan oleh service Windows yang bermasalah atau timing service yang delay.


2. Instal ulang graphic (display) adapter

Fungsi dari graphic adapter sendiri adalah menampilkan output hasil processing ke layar monitor, dan rendering, meski sekilas tak ada kaitan langsung antara blank screen dengan kerja graphic adapter, tetapi sebenarnya masalah pada driver graphic bisa mengakibatkan sistem menjadi unstable dan menjadi penyebab blue screen  misalnya. Adapun untuk instal ulang graphic adapter langkahnya adalah sbb :

Nyalakan laptop;

Masuk ke safe mode seperti langkah diatas;

Tekan tombol keyboard CTRL + ALT + DELETE untuk memunculkan Task Manager;

Pada jendela taskmanager, tekan CTRL + N atau klik menu File > New kemudian ketik explorer.exe dan tekan Enter;

Setelah Windows Explorer muncul, klik ikon Computer pada panel kiri, dan klik Manage;

Pada jendela Computer management yang muncul, pada panel kiri, klik Device Manager;

Pada panel kanan, expand section Display adapter, klik kanan pada driver yang muncul dan klik Properties;

Pada jendela yang muncul, klik Update driver software, ikuti langkah yang diberikan, hingga selesai;

Pada tahap ini, jika instalasi driver sebelumnya memang memerlukan CD-ROM driver bawaan adapter atau dapat unduh dari internet, maka ada opsi untuk mengambil source instalasi driver dari disk;

Setelah proses selesai, restart laptop, jika masalah driver graphic selesai, seharusny Windows 7 sudah masuk Desktop.

Cara Pertama

Anda coba menekan tombol dengan logo Windows + P, setelah itu tekan tombol yang bergambar panah.

Kemudian lihat dan perhatikan pada windows laptop Anda, apakah kursor di layar keluar, apabila hal tersbut terjadi, tekan dan tahan tombol Spasi dan juga ketik pasword atau sandi, lalu tekan Enter.

Cara Kedua

Lepaskan atau copot Baterai Laptop Anda.

Cabut Casan Laptop Anda.

Tekan dengan menahan tombol Power sampai 10 detik.

Copt tombol Powernya.

Setelah itu pasang semua perangkat yang dilepaskan tadi, kemudian nyalakan kembali.

Cara Ketiga

Pada saat laptop Anda windows nya berlayar hitam atau blackscreen Anda coba dengan cara ini yaitu : Tahan dan tekan tombol ALT + DEL + CTRL dengan bersamaan, yang bertujuan untuk membuka Task Manager, apabila sudah terbuka, Klik File ~ New ~ lalu ketik explorer.exe

Apabila berhasil dengan selesai maka Anda harus melakukan System pada windows untuk di Restore dan aplikasi yang membuat tidak kompatible segera di Uninstal.

Cara Keempat

Yang pertama pada pengaturan windows Background Show harus aktif (Anda bisa lihat di settingan)

Setelah itu masuk ke setting dengan cara menekan tombol logo Window + I atau juga bisa ketik Setting di Menu Start

Kemudian Anda pindahkan jendelanya di Ease of access ~ Other Options. Dengan cara seperti ini yaitu buka Settings — > Lalu Ease of access — > dan Other Options.

Berikutnya harus menyala pada sklar Windows Background Show

Lalu Anda harus memilih thema sesuai keinginan Anda

Yang terakhir Anda harus selesaikan terlebih dahulu kerjaannya, setelah itu keluar dengan ketik Sign out dari laptop windows 10, setelah itu masuk kemabli dnegan Sign In, maka dengan ini layar dekstop sobat akan kembali normal.

19. bagaimana caranya mengatasi sinyal wifi yang lemah di windows 10


Klik Start – Run, lalu ketik “CMD”. Atau bisa langsung search “CMD”.

Klik kanan, Run As Administrator.

Setelah muncul jedela CMD, ketikkan perintah “netsh interface tcp show global”.

Tekan Enter.

Perhatikan pada “Receive Window Auto-Tuning Level.” Statusnya “Normal/Enable”.

￼Window Auto-Tuning (Normal)

Nah tugas kita adalah disable Window Auto-Tuning tersebut.

Ketikkan perintah ” netsh int tcp set global autotuninglevel=disabled ” Lalu enter.

Jika muncul tulisan “OK” berarti sukses.

Windows Auto-Tuning sudah disable.
