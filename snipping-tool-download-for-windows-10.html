1. apa kegunaan snipping tool


Bagi sobat yang menggunakan Sistem operasi Windows 7, mungkin belum banyak yang tau apa manfaat Snipping Tool. Windows 7 sendiri mempunyai banyak kelebihan dari sistem operasi sebelumnya seperti Windows XP, Vista atau yang lainnya. Salah satu kelebihan tersebut adalah adanya Sipping Tool.Apa itu Snipping Tool ??. Kalo dari artinya Snipping Tool = Alat pemotong. Fungsinya adalah untuk memotong, memotong apa ya??hehe... Manfaat Snipping Tool pada Windows 7 adalah untuk mengambil screen Shoot. Jadi dengan adanya tool ini anda tidak perlu lagi menggunakan software screen shoot yang fungsinya kurang lebih sama dengan Tool bawaan windows 7 ini.
Snipping Tool biasanya dapat diakses dari menu start-->Accessories-->Snipping Tool. Setelah terbuka, klik New untuk mulai mengambil screen shoot.Selanjutnya cari posisi untuk mulai membuat screen shoot,,misalnya dari pojok kiri atas lalu drag ke pojok kanan bawah. Pembahasan:

Media creation tool adalah sebuah program yang didesain oleh MIcrosoft untuk mempermudah aktivitas yang berkaitan dengan instalasi Windows yang diantaranya:

Melakukan upgrade ke versi Windows terbaru secara langsungMembuat media instalasi pada USB atau DVDMengunduh file (ISO) instalasi Windows saja

Dalam hubungannya dengan pertanyaan yang kamu ajukan upgrade dari Windows versi lama ke Windows 10 menggunakan media creation tool akan memberikan keuntungan seperti

Bisa memilih antara arsitektur 32 dan 64 bitProses/langkahnya yang mudah dimengertiAman karena jika yang dipilih adalah upgrade maka kamu akan memiliki data cadangan dari Windows terdahulu.
