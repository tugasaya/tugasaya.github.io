5. untuk mengambil gambar di Windows media player dapat menggunakan perintah A. improt video B. improt picture C. improt audio or musik D. save to my computer


Untuk mengambil gambar di Windows media player dapat menggunakan perintah...

A. import video

B. import picture V

C. import audio or musik

D. save to my computer

Pembahasan

Cara Mengambil Gambar Dari Video Windows Media Player, maka saya akan mencontohkannya dengan menggunakan Windows Media Player.

1. Buka video di Windows Media Player

Untuk membuka video di Windows Media Player (WMP) cukup mudah, yaitu tinggal Anda klik kanan pada video yang akan dibuka, lalu pilih Open With > Widnwos Media Player.

2. Tekan tombol Printscren

Saat video berjalan silahkan Anda tekan tombol printscreen yang terdapat di keyboard Anda, setiap keyboard memiliki penamaan tombol yang berbeda, untuk laptop ASUS tombol tersebut ditulis dengan prt sc sysrq untuk keyboard lain ada juga yang ditulis prt scr. Intinya cari tombol yang ada tulisan prt.

3. Buka aplikasi Paint

Anda bisa membuka paint atau aplikasi edit foto lainnya, seperti photoshop atau photoscape, bebas. Selanjutnya silahkan Anda tekan CTRL + V atau bisa juga dengan membuka Menu lalu pilih Paste. Di langkah ini Anda juga bisa melakukan cropping atau pemotongan gambar.

4. Simpan gambar hasil Printscreen tersebut

Langkah akhir adalah menyimpan gambar yang sudah kita ambil dari video dengan cara menekan tombol printscreen tadi, dan untuk menyimpannya juga sangat mudah, yang pertama bisa dengan cara klik CTRL + S, atau cara yang kedua adalah dengan manual dengan membuka File > Save as > JPEG picture.

Dengan tujuan untuk mengambil gambar (screenshot) video pada suatu layar komputer yang ditonton menggunakan Windows Media Player (WMP), biasanya seseorang akan langsung menggunakan tombol “Print Screen” di keyboard. Namun yang terjadi ketika hasil print screen tersebut coba di paste ke sebuah editor image misalnya MS Paint atau Photoshop akan mendapatkan hasil nihil alias screenshot gambar dari video tersebut tidak akan tampak sama sekali.

Sebenarnya dengan menekan tombol Print Screen untuk mengambil gambar video pada aplikasi player ini sudah benar adanya, hanya saja Windows Media Player tidak menawarkan secara default fungsi ini didalam aplikasinya. Jadi untuk itu agar tujuan anda dapat berhasil, maka fungsi tersebut haruslah terlebih dahulu di atur.

Agar dapat melakukan print screen di Windows Media Player, maka terlebih dahulu anda harus me non aktifkan fungsi Overlays nya dengan cara:

1. Buka apliksi WMP, lalu akses menu “Tools > Options”. (tips jika menu tools tidak muncul, klik kanan title bar WMP, nanti menunya pasti akan nongol).

2. Pilih tab “Performance” lalu klik tombol “Advance”.

3. Hilangkan centang pada bagian “Use Overlays” dan klik tombol “Ok” untuk menyimpan settingannya.

Setelah setting telah diatur, silahkan buka video apapun melalui Windows Media Player lalu pause video nya, berikutnya tekan tombol keyboard “Print Screen” atau “Alt + Print Screen” dan paste ke image editor. Hasilnya pasti akan berhasil.

Mic Internal Kamera :

Suara mengema

Suara lebih bebas noise

Suara lebih kecil

Suara yang ditangkap menyeluruh

Mic Eksternal :

Suara lebih jelas

Suara noise tertangkap *tergangung tipe mic, kalau seri tinggi mic ada fitur peredam noise

Mikrofon eksternal tidak harus yang dipasang di atas kamera, tetapi bisa model lain sesuai kebutuhan, misalnya mic clip on, mic boom, atau mic condenser vokal, tergantung kebutuhan dan budget.
