KPK = Kelipatan persekutuan terkecil.

Masukkan semua angka, jika ada angka yang sama pilih angka terbesar.

FPB = Faktor persekutuan terkecil.

Pilih angka yang sama, jika ada angka yang sama pilih angka terkecil.

FPB dan KPK dari 20 dan 25

20 = 2² x 5

25 = 5²

FPB = 5

KPK = 2² X 5² = 4 X 25 = 100

FPB = 2 X 3 X 5 = 6 X 5 = 30

KPK = 2² X 3² X 5 = 4 X 9 X 5 = 180

32 = 2⁵

42 = 2 x 3 x 7

FPB = 2

KPK = 2⁵ x 3 x 7 = 32 x 3 x 7 = 672

FPB = 2³ = 8

KPK = 2⁴ x 3 = 16 x 3 = 48
