4. Limit Fungsi Tak Hingga​


Jawab:

1- Mari kita evaluasi limit yang diberikan:

lim ( (2x-1) - akar (4x^2 + 10x -5) ) saat x mendekati nilai yang tidak ditentukan.

Pertama-tama, kita dapat mencoba untuk menyederhanakan ekspresi di bawah akar. Kita dapat mengalikan dan membagi dengan (2x + 1) pada bagian dalam akar:

lim ( (2x-1) - akar (4x^2 + 10x -5) )

= lim ( ((2x-1) - akar (4x^2 + 10x -5)) * ( (2x+1) / (2x+1) ) )

= lim ( ( (2x-1)*(2x+1) - (4x^2 + 10x -5) ) / ( 2x+1 + akar (4x^2 + 10x -5) ) )

Kemudian, kita dapat menyederhanakan ekspresi pada pembilang dengan mengurangi 4x^2 pada kedua sisi:

= lim ( ( -1 - 10x + 5 ) / ( 2x+1 + akar (4x^2 + 10x -5) ) )

= lim ( (-10x + 4) / (2x+1 + akar(4x^2 + 10x -5) ) )

Sekarang, kita dapat menggunakan aturan substitusi untuk menentukan nilai limit saat x mendekati nilai yang tidak ditentukan. Kita dapat mencoba untuk mengalikan dan membagi dengan faktor tertinggi di pembilang dan penyebut:

lim ( (-10x + 4) / (2x+1 + akar(4x^2 + 10x -5) ) )

= lim ( -10 (x - 0.4) / ( 2(x+0.5) + akar( 4(x+0.5)^2 - 9.75 ) ) )

Ketika x mendekati nilai yang tidak ditentukan, x - 0.4 mendekati 0 dan x + 0.5 mendekati nilai yang tidak ditentukan. Sehingga, limit dari fungsi saat x mendekati nilai yang tidak ditentukan adalah:

lim ( -10 (x - 0.4) / ( 2(x+0.5) + akar( 4(x+0.5)^2 - 9.75 ) ) )

= lim ( -10 (x - 0.4) / ( 2(x+0.5) + akar( 4(x+0.5)^2 ) ) )

Karena x + 0.5 mendekati nilai yang tidak ditentukan, akar (4(x+0.5)^2) akan mendekati 2(x+0.5) saat x mendekati nilai yang tidak ditentukan. Sehingga limit dari fungsi saat x mendekati nilai yang tidak ditentukan adalah:

lim ( -10 (x - 0.4) / ( 2(x+0.5)

2- Mari kita evaluasi limit yang diberikan:

lim (2x + sin x) / (x - cos 2x) saat x mendekati nilai yang tidak ditentukan.

Karena kita sedang mencari limit saat x mendekati nilai yang tidak ditentukan, kita perlu memeriksa apakah fungsi di atas memiliki asimtot vertikal atau horizontal.

Jika terdapat asimtot vertikal, maka limit akan bernilai tak terdefinisi (biasanya berupa tak hingga positif atau negatif). Jika terdapat asimtot horizontal, maka limit akan bernilai konstan.

Untuk memeriksa apakah terdapat asimtot vertikal atau horizontal, kita perlu memeriksa perilaku fungsi di atas saat x mendekati tak hingga positif dan negatif.

Periksa saat x mendekati tak hingga positif:

lim (2x + sin x) / (x - cos 2x) saat x mendekati tak hingga positif.

Kita dapat membagi setiap suku pada pembilang dan penyebut dengan x untuk mendapatkan bentuk limit tak hingga atas tak hingga:

lim (2x / x) + (sin x / x) / (x / x) - (cos 2x / x) saat x mendekati tak hingga positif.

Dalam limit tak hingga atas tak hingga, suku yang paling dominan adalah 2x pada pembilang dan x pada penyebut. Sehingga limit ini akan mendekati tak hingga positif.

Periksa saat x mendekati tak hingga negatif:

lim (2x + sin x) / (x - cos 2x) saat x mendekati tak hingga negatif.

Kita dapat membagi setiap suku pada pembilang dan penyebut dengan x untuk mendapatkan bentuk limit tak hingga atas tak hingga:

lim (2x / x) + (sin x / x) / (x / x) - (cos 2x / x) saat x mendekati tak hingga negatif.

Dalam limit tak hingga atas tak hingga, suku yang paling dominan adalah 2x pada pembilang dan -cos(2x) pada penyebut. Karena cos(2x) memiliki nilai absolut yang terbatas, maka limit ini akan mendekati tak hingga negatif.

Karena terdapat limit tak hingga saat x mendekati tak hingga positif dan negatif, maka tidak ada asimtot horizontal pada fungsi di atas. Oleh karena itu, tidak terdapat limit saat x mendekati nilai yang tidak ditentukan pada fungsi di atas. Dengan demikian, nilai limit tersebut tidak dapat ditentukan.

PEMBAHASAN

Limit fungsi merupakan keadaan dari suatu fungsi saat mendekati suatu titik. Misalnya fungsi f(x) tidak terdefinisi saat x = a namun bernilai L saat mendekati a. Secara matematis dapat dituliskan menjadi:

[tex]\boxed{\boxed{\sf{\lim_{x \to a}f(x) = L}}}[/tex]

ㅤ

Teorema Limit

Berikut beberapa teorema limit utama.

1. [tex]\displaystyle{\sf{\lim_{x \to a}k = k}}[/tex]

2. [tex]\displaystyle{\sf{\lim_{x \to a} {k. \: x}^{n} = k. \: {a}^{n}}}[/tex]

3. [tex]\displaystyle{\sf{\lim_{x \to a}k. \: f(x) = k. \: \lim_{x \to a} \: f(x)}}[/tex]

4. [tex]\displaystyle{\sf{\lim_{x \to a}f(x) \pm g(x) = \lim_{x \to a}f(x) \pm\lim_{x \to a}g(x)}}[/tex]

5. [tex]\displaystyle{\sf{\lim_{x \to a}f(x) \times g(x) = \lim_{x \to a}f(x) \times \lim_{x \to a}g(x)}}[/tex]

6. [tex]\displaystyle{\sf{\lim_{x \to a} \dfrac{f(x)}{g(x)} = \dfrac{\displaystyle{\sf{\lim_{x \to a}f(x)}}}{\displaystyle{\sf{\lim_{x \to a}g(x)}}}}}[/tex]

7. [tex]\displaystyle{\sf{\lim_{x \to a}{\left[f(x)\right]}^{n} = {\left[\lim_{x \to a}f(x)\right]}^{n}}}[/tex]

8. [tex]\displaystyle{\sf{\lim_{x \to a}\sf{\sqrt[\sf{n}]{\sf{f(x)}}}=\sqrt[\sf{n}]{\displaystyle{\sf{\lim_{x\to a}f(x)}}}}}[/tex]

ㅤ

ㅤ

Diketahui:

[tex]\sf{\lim\limits_{x\to2}{2x}^{2}+x+2}[/tex]

ㅤ

Ditanyakan:

Nilai limit fungsinya jika menggunakan teorema limit adalah …

ㅤ

Jawab:

[tex]\begin{array}{lll}\sf{\lim\limits_{x\to2}{2x}^{2}+x+2}&=&\sf{\lim\limits_{x\to2}{2x}^{2} +\lim\limits_{x\to2}x+\lim\limits_{x\to2}2}\\\\&=&\sf{{2.2}^{2}+2+2}\\\\&=&\sf{2.4+4}\\\\&=&\sf{8+4}\\\\&=&\sf{12}\end{array}[/tex]

Jadi nilai limit dari [tex]\sf{\lim\limits_{x\to2}{2x}^{2}+x+2}[/tex] dengan menggunakan teorema limit adalah [tex]\boxed{\sf{12}}.[/tex]

12. 1. tentukan perbedaan limit fungsi aljabar dan limit fungsi trugonometri​


Perbedaan dari kedua limit tersebut adalah jenis fungsi yang akan digunakan. Berbeda karena, salah satu limit menerapkan fungsi aljabar sebagai bahan pendekatan dan yang satunya lagi menerapkan fungsi trigonometri sebagai bahan pendekatan.

PEMBAHASAN

Limit Fungsi adalah sebuah konsep dalam matematika yang menguji sebuah fungsi apakah menghasilkan nilai jika fungsi tersebut mendekati suatu nilai tertentu. Limit terbagi menjadi 3, diantaranya Limit Fungsi Aljabar, Limit Fungsi Tak Hingga, dan Limit Fungsi Trigonometri.

Masing masing limit memiliki cara pengerjaan yang agak berbeda dari yang lainnya. Mulai dari fungsi aljabar, jika nilai substitusinya sama dengan 0/0 kita harus menggunakan cara faktor, kali akar sekawan, atau turunkan fungsinya. Ada banyak sekali cara pengerjaannya, sobat bisa simak pembahasan lebih lanjut dibawah ini!

SUBSTITUSI

Berlaku untuk semua jenis Limit, Kita bisa pakai cara ini jika hasil substitusi sebuah fungsi tidak sama dengan 0/0.

FAKTOR

Berlaku untuk semua jenis limit, kita bisa pakai cara ini jika hasil substitusi sebuah fungsi terhadap nilai pendekatan sama dengan 0/0 serta pembilang dan penyebut memiliki satu faktor yang sama.

DIFERENSIALKAN FUNGSINYA

Berlaku untuk semua jenis limit, jika hasil substitusi sebuah fungsi menghasilkan 0/0. Kita bisa menurunkan derajat fungsinya dengan menggunakan konsep diferensial atau turunan.

KALI AKAR SEKAWAN

Berlaku untuk limit yang memiliki fungsi rasional atau bilangan akar, kita bisa menggunakan cara ini jika fungsi merupakan bilangan akar yang dapat menghasilkan 0/0 jika di substitusikan.

IDENTITAS TRIGONOMETRI

Berlaju untuk limit yang merupakan fungsi trigonometri, jika hasil substitusi sama dengan 0/0. Kita bisa menyederhanakan sebuah fungsi trigonometri dengan menerapkan identitas atau rumus trigonometrinya. Sampai ke titik dimana fungsi tersebut sudah di level yang paling sederhana.
