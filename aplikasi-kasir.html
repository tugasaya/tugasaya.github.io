3. Bagaimana caranya membuat program aplikasi yang dipai d supermarket contohnya seperti di kasir????


program/aplikasi yang dipakai di supermarket/minimarket untuk kasir biasanya ada 2 jenis, yang pertama berjalan di Disk Operating System(DOS), yang kedua berjalan di Operating System(OS).

Jika ingin membuat yang berjalan pada mesin DOS(Mesin kasir langsung), anda bisa menggunakan bahasa pemrogramman yang mendukung DOS seperti C, Cobol, Fortran dll.

Jika ingin membuat yang berjalan di OS(Menggunakan komputer desktop), anda dapat menggunakan bhs pemrogramman berbasis Java, .NET, C# dll

[jawaban]

Kata kunci: Visual Basic

• Penjelasan:

Bagaimana cara menghitung rp dengan banyak barang. Dalam form kasir sederhana dalam aplikasi visual basic dan hasilnya juga menggunakan mata uang rp?

Intinya, anda menentukan harga barang tersebut, dan masukkan dalam sebuah variabel. Jika user mengambil beberapa barang (misal 2), tinggal kalikan variabel tersebut sebanyak yang diambil user.

Contoh:

// harga apel Rp. 10.000;
int harga_apel = 10000;

// misal user mangambil 2 apel;
int user = 2;

// harga total 
int harga_total = harga_apel * user;

// output
print ("Harga total = Rp. " + harga total);
