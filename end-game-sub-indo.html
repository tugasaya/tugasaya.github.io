22. = Private Sub Commandl Click(). nilai = Text1.Text If nilai >= 88 Then If nilai >= 95 Then MsgBox "Nilai A +" Else MsgBox. "Nilai A-" End If ElseIf nilai >= 68 Then MsgBox "Nilai B" ElseIf nilai >= 58 Then MsgBox "Nilai C" Else MsgBox. Tidak lulus" End If End Sub 10. Jika nilai adalah 88, maka output program adalah .... a. Nilai A+ C. Nilai B b. Nilai A- d. Nilai C​


Jawaban:

b. Nilai A-

Penjelasan:

Dirapikan dulu source-code-nya.

Private Sub Commandl Click()

   nilai = Text1.Text

   If nilai >= 88 Then

       If nilai >= 95 Then

           MsgBox "Nilai A +"

       Else

           MsgBox "Nilai A-"

       End If

   ElseIf nilai >= 68 Then

       MsgBox "Nilai B"

   ElseIf nilai >= 58 Then

       MsgBox "Nilai C"

   Else

       MsgBox "Tidak lulus"

   End If

End Sub

Karena nilai adalah 88, maka:

kondisi nilai >= 88 terpenuhi. Oleh karena itu, baris program di dalamnya dieksekusi, yaitu:

If nilai >= 95 Then

   MsgBox "Nilai A +"

Else

   MsgBox "Nilai A-"

End If

Di sini, kondisi nilai >= 95 tidak terpenuhi. Oleh karena itu, yang akan dieksekusi adalah baris program dalam blok "else", yaitu:

MsgBox "Nilai A-"

∴ Jadi, output program adalah sebuah message box menampilkan tulisan: Nilai A-.
