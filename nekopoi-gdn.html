Pertanyaan:

1). 1 liter = .... dm³

Jawab:

1). 1 dm³

Penjelasan:

Terdapat hubungan antara dua jenis satuan volume, yaitu:

• 1 liter = 1 dm³

• 1 ml = 1 cm³

• 1 ml = 1 cc

• 1 cm³ = 1 cc

.

Hasil akhir

Jadi,

1). Hasil dari 1 liter jika diubah menjadi satuan volume dm³ adalah 1 dm³.

Jawaban

∠B = 52°

∠C = 78°

Penyelesaian Soal

Ubah perbandingan menjadi,

∠B : ∠C = 2x : 3x

———

A + B + C = 180

50 + 2x + 3x = 180

50 + 5x = 180

5x = 130

x = 26

Maka,

∠B = 2x

= 2(26)

= 52°

∠C = 3x

= 3(26)

= 78°

Panjang sudut keseluruan segitiga = 180 °

∠A = 50

∠B : ∠C = 2 : 3

jadi bisa dibilang kalau

∠B = 2 × ....... atau 2 × x = 2x

∠C = 3 × ..... juga atau 2 × x = 3x

keseluruan segitiga = 180°

180° = ∠A + ∠B + ∠C

180° = 50° + 2x + 3x

180° = 50° + 5x

180° - 50° = 5x

130° = 5x

130/5 = x

26 = x

Yang di cari

B = 2x

= 2 × 26 = 52°

C = 3x

= 3 × 26 = 78°
