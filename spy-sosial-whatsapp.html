10. spyhasil dari 78% +0,4-tiga per empat​


Penjelasan dengan langkah-langkah:

78% + 0,4 - 3/4

= 78/100 + 4/10 - 3/4

= (100:100×78) + (100:10×4) - (100:4×3)

= 78/100 + 40/100 - 75/100

= 43/100 => pecahan biasa

= 43% => persen

= 0,43 => desimal
