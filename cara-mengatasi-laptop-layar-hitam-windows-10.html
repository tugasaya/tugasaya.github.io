30. cara mengatasi ram usage di laptop windows 10 jangan asal ​


Jawaban:

Penyebab RAM Usage Tinggi

Sebelum kamu tahu cara untuk mengatasinya, kamu harus tahu dulu penyebab yang mengakibatkan pemakaian RAM tinggi.

Beberapa penyebabnya:

Kapasitas RAM yang memang kecil.

Pemakaian aplikasi secara bersamaan dalam satu waktu.

Terinfeksi virus

Menggunakan background atau tema dengan size besar.

Menggunakan beberapa efek dan layanan sistem operasi.

Terlalu banyak memasang aplikasi.

cara mengatasi:

Cek dan Stop Startup Service di Task Manager

Service ini berfungsi untuk menjalankan program-program pada saat sistem baru aktif. Ini perlu di-cek, agar pemakaian RAM bisa berkurang.

Berikut langkah-langkahnya:

Pertama klik kanan pada Taskbar lalu pilih Task Manager

Masuk ke tab Startup

Kemudian pilih program mana saja yang tidak terlalu penting

Ubah status program yang “Enabled” menjadi “Disabled”.
