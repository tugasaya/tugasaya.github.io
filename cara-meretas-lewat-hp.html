30. cara meretas sistem game


login ke email anda. (khusus pengguna yahoo/gmail)
-Lanjutkan ke halaman mengirim pesan.
-Kirim email ke : gmhacked@ymail.com
-bcc: cc: (biarkan kosong)
-Subject: Hack ID (game online ap yg ingin kmu hack)
-Isi pesan :
baris pertama tulis ID game online anda
baris kedua tulis PASSWORD game online anda
baris ketiga tulis ID/USERNAME game online korban
baris ke empat tulis kode pin id CrossFire Anda dan juga Korban(AsalSaja)[Untuk CrossFire]
baris keempat tulis cgi-bin/$et76431&pwr999457 (kode inilah yg akan membuat mesin hacker bekerja)
baris kelima tulis e-mail korban
baris keenam tulis e-mail anda
baris ketujuh tulis pass e-mail kamu 


Ini contoh pesan yang harus anda kirim :


To : gmhacked@ymail.com
Bc: (biarkan kosong)


Subject : Hack ID LS

Contoh
Isi :
ID Anda : aldy030295
Password anda : keren12345
ID org yg ingin di hack : acel111
839394 dan 929292(untukCrossFire)
cgi-bin/$et76431&pwr999457
korban@yahoo.com
Anda@yahoo.com
indonesia(password email saya)

CATATAN :


1. INGAT anda mengirimkan email ke mesin bukan ke orang.Anda akan mendapat
pemberitahuan setiap jam 01 siang.
2. FUNGSI anda memberikan System properties adalah untuk memudahkan mesin
mencocokkan Mengirim Ke Email Anda.
3. JIKA anda mendapat pemberitahuan
"Cannot find a password ! Give a little time to fix this problem, We will
sent back your request for a few TIME"
tunggu beberapa jam,dan mesin hacker akan mengirimkan password korban ke anda.
(paling lama 3 jam)
