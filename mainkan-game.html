11. apakah bermain game boleh dalam Islam?​


Jawaban:

Menurut pandangan umum dalam Islam, bermain game boleh-boleh saja asalkan tidak melanggar syariat Islam dan tidak membuat pemainnya melupakan kewajiban-kewajiban agamanya.

Beberapa hal yang harus diperhatikan dalam bermain game adalah:

Tidak melibatkan unsur-unsur haram seperti judi, seksual, dan kekerasan.Tidak membuang-buang waktu secara berlebihan dan mengganggu kegiatan lain yang lebih penting.Tidak mempengaruhi sikap dan tingkah laku pemain secara negatif.

Namun, meskipun bermain game tidak secara langsung melanggar syariat Islam, tetap saja lebih baik jika pemain memprioritaskan kegiatan-kegiatan yang lebih bermanfaat dan memberikan nilai positif bagi diri mereka sendiri maupun masyarakat.

25. apakah makna dari kata gime online,game developer,maniak game,pembuat game,bermain game,pengembang game


game online = Game/ Permainan yang dimainkan melalui situs tertentu atau harus terkoneksi internet
game developer = Seseorang/ perusahaan yang bertugas mengembangkan game/ permainan, Entah secara fitur atau tampilan
maniak game = Seseorang yang tergila gila pada suatu game sampai 'menghalalkan segala cara' untuk bisa memiliki/memainkan game tersebut
bermain game = memainkan permainan
pengembang game = Game developer sebenarnya.... Hanya kadang penggunaannya berbeda. Kalau game developer itu seseorang/peruahaan, biasanya pengembang game itu perusahaan.
