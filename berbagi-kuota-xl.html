16. Nilai x yang memenuhi persamaan l2-xl²-l2-xl=3l2-xl+5 adalah​


Penjelasan dengan langkah-langkah:

|2 - x|² - |2 - x| = 3|2 - x| + 5

misal |2 - x| = p, maka :

p² - p = 3p + 5

p² - p - 3p - 5 = 0

p² - 4p - 5 = 0

(p + 1)(p - 5) = 0

p = -1 atau p = 5

• untuk p = -1

|2 - x| = -1 (tidak memenuhi, karena nilai mutlak tidak mungkin bernilai negatif)

• untuk p = 5

|2 - x| = 5

2 - x = -5 atau 2 - x = 5

-x = -5 - 2 atau -x = 5 - 2

-x = -7 atau -x = 3

x = 7 atau x = -3

Jadi, HP = {-3, 7}

25. jika B = { xl -2 < × < 2, × bilangan bulat} , maka banyak himpunan Bagian dari B adalah​


Jawaban : 8

PembahasanHimpunan Bagian

adalah himpunan" yg terbentuk dr suatu himpunan. Sehingga anggota dr himpunan bagian juga anggota dari himpunan tersebut.

Rumus nya : 2^n (2 pangkat n)

Keterangan

n : banyak nya anggota himpunan

Pembahasan Soal

B = { xl -2 < × < 2, × bilangan bulat}

B = {-1, 0, 1}

Banyak anggota himpunan B adalah 3

Berarti banyak himpunan bagian nya adalah

2^n = 2³ = 2×2×2 = 8

Jadi, banyak himpunan bagian nya ada 8
