5. 5.def top ()top = len(stack) -1if top<0print "tidak terdefinisi"elseprint stack[top]maksud dari perintah/instruksi tersebut adalahâ€‹


Jawaban:

Penjelasan:

Sepertinya ini perintah untuk mencek apakah stack memiliki elemen atau tidak.

Stack memiliki elemen atas dan bawah (tapi lebih signifikan elemen atas), jika panjang stack = 1, maka elemen atas = bawah.

Pada kode tersebut method top() pertama akan mengurangi panjang variabel list stack sebesar 1 dan menyimpannya ke variabel top.

Lalu jika top bernilai negatif, program akan mencetak "tidak terdefinisi". Yang artinya stack belum memiliki elemen (dan akibatnya elemen teratas belum terdefinisi).

Jika tidak maka program akan mencetak elemen dari stack[] pada indeks berupa nilai variabel top (mencetak elemen teratas pada stack[]).

Pembahasan

Permainan bulu tangkis adalah salah satu olahraga bola besar yang dapat dimainkan secara tunggal atau beregu. Tujuan dari permainan bulu tangkis adalah untuk mematikan bola di area permainan lawan dan memenangkan pertandingan. Permainan bulu tangkis dimainkan di lapangan tertutup yang terhindar dari hembusan angin.

Dalam permainan bulu tangkis, setiap pertandingan dimainkan dalam 2 atau 3 babak. Jika pemain atau regu berhasil memenangkan 2 set awal, maka dinyatakan sebagai pemenang dan pertandingan berhasil. Namun jika kedua pemain atau regu imbang di 2 set awal, maka pertandingan dilanjutkan pada set ke 3 sebagai set penentuan.

PEMBAHASANDegree of Comparison

Yaitu bentuk kata sifat yang digunakan untuk membandingkan atau menunjukkan tingkat atau derajat yang berbeda terhadap suatu hal berjumlah dua atau lebih.

Comparative Degree ditandai dengan adanya kata more sebelum kata sifat dan adjective yang berakhiran -er.

Sedangkan Superlative Degree ditandai dengan adanya kata most atau adjective yang berakhiran -est.

Examples:

(Positive - Comparative - Superlative)

Clean - Cleaner - Cleanest

Bersih - Lebih bersih - Paling bersih

Dirty - Dirtier - Dirtiest

Kotor - Lebih kotor - Paling kotor

Fast - Faster - Fastest

Cepat - Lebih cepat - Paling cepat

Slow - Slower - Slowest

Lambat - Lebih lambat - Paling lambat

Jawab:

Mari kita sebut jumlah premium set yang terjual sebagai x.

Dari soal, kita tahu bahwa jumlah simple set dan kids set yang terjual 30 lebih banyak dari setengah premium set yang terjual. Dengan kata lain,

jumlah simple set dan kids set yang terjual = 1.5x + 30

Kita juga tahu bahwa jumlah simple set yang terjual 5 lebih banyak dari 4 kali jumlah kids set yang terjual. Dengan kata lain,

jumlah simple set yang terjual = 4 * jumlah kids set yang terjual + 5

Kita bisa menggunakan kedua persamaan di atas untuk menyelesaikan masalah ini. Kita tahu bahwa total penjualan dari semua jenis rice box adalah Rp. 89.820.000,00, sehingga kita dapat menuliskan persamaan:

33.000x + 24.000(1.5x + 30) + 9.000(4x + 1) = 89.820.000

Setelah menyelesaikan persamaan tersebut, kita akan mendapatkan x = 1.800.

Kita dapat menggunakan nilai x tersebut untuk mencari jumlah simple set yang terjual:

jumlah simple set yang terjual = 4 * jumlah kids set yang terjual + 5 = 4 * (1.800 - 30)/5 + 5 = 573

Jadi, jumlah simple set yang terjual adalah 573.
