Dik :

a = 4

b = 3

Dit

Un = 22 = ?

Peny :

Barusan jika kursi sebanyak 22 buah

Un = a + (n-1) × b

22 = 4 + (n-1) × 3

22 = 4 + 3n - 3

22 = 3n + 1

3n = 22 - 1

3n = 21

n = 21/3

n = 7

Penyelesaian- Diket :

• a / U1 = 2

• b = 2

- Dit :

• U7 = ?

• Un = a + (n - 1) × b

• U7 = 2 + (7 - 1) × 2

• U7 = 2 + (6) × 2

• U7 = 2 + 12

• U7 = 14 anak
