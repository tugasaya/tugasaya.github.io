Pembahasan

POLA PERSEGI PANJANG

Pola bilangan adalah barisan bilangan yang memiliki pola tertentu. Selain pola barisan aritmatika dan pola barisan geometri ada beberapa pola barisan yang umum, yaitu:

Barisan Fibonacci merupakan jumlah dua suku dari barisan sebelumnya. Misalkan 1, 1, 2, 3, 5, 8, ...Barisan bilangan asli = 1, 2, 3, 4, ... = nBarisan bilangan genap = 2, 4, 6, 8, ... = 2nBarisan bilangan ganjil = 1, 3, 5, 7, 9, ... = 2n - 1Barisan persegi atau kuadrat = 1, 4, 9, 16, ... = n²Barisan persegi panjang = 1 × 2, 2 × 3, 3 × 4, ... = n × (n + 1)

Ditanyakan

Banyak persegi pada pola barisan persegi panjang ke-22?

Penjelasan

Pola ke-22, artinya n = 22

[tex]U_n[/tex] = n × (n + 1)

[tex]U_{22}[/tex] = 22 × (22 + 1)

19. Tentukan jumlah bilangan pada pola persegi panjang suku ke-22 2+4+6+8+...+ suku ke-22


Jumlahnya 506.

Pembahasan

Barisan Aritmatika adalah  suatu barisan dengan selisih antara dua suku yang berurutan selalu tetap.

Rumus : [tex]\boxed{~Un~=~a+(n-1)b~}[/tex]

Deret Aritmatika adalah jumlah suku – suku barisan aritmatika

Rumus : [tex]\boxed{~Sn~=~\frac{n}{2}(a+Un)~}[/tex]

Atau [tex]\boxed{~Sn~=~\frac{n}{2}(2a+(n-1)b)~}[/tex]

Untuk Barisan Aritmatika bertingkat Rumus Umumnya adalah

[tex]\boxed{~Un~=~an^2+bn+c~}[/tex]

a = 2

b = 2

n = 22

Ditanya:

Jumlah bilangan sampai pola ke-22

Jawab:

Sn = [tex]\frac{n}{2}[/tex] (2a + (n-1)b)

S₂₂ = [tex]\frac{22}{2}[/tex] (2.2 + (22-1)2)

     = 11 (4 + 42)

     = 11 (46)

     = 506
