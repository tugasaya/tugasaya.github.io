24. textbox juga pada dasarnya adalah sehingga dapat diformat tampilannya Bila kita ingin mewarnai garis pada textbox maka menggunakan menu​


Untuk menebalkan garis dan bingkai pada Text Box klik ikon Weight.

Pembahasan

Text box adalah bentuk kotakan yang dapat kita isi dengan text.

Cara untuk menggunakannya adalah sebagai berikut :

1. Klik Insert, kemudian klik Text Box, maka akan muncul beberapa Pilihan Text Box, pilihlah sembarang karena kita dapat mendesain Text Box belakangan.

2. Setelah itu Text Box akan muncul dalam lembar kerja Microsoft Word kita, di dalam Text Box tersebut ada sebuah tulisan, disitulah kita akan menulis tulisan sesuai kebutuhan.

3. Untuk memperbesar ukuran Text Box sesuai dengan keinginan, caranya adalah dengan menunjuk point mouse ke arah lingkaran kecil berwarna orange dan mendrag mouse sesuai dengan ukuran yang kita inginkan.

4. Untuk menggeser Text Box sesuai kehendak, caranya dengan menunjuk pointer mouse di daerah kotakkan garis, sehingga menggambarkan tanda +, kemudian drag dan geser Text Box sesuai dengan kebutuhan.

5. Untuk merubah warna, garis, dan bentuk Text Box dengan mengaktifkan Text Box, kemudian Pilih Format kemudian Pilih Shape Fill. Berikut menu-menu yang tersedia :

Menu Picture : untuk mengisi Text Box dengan gambar dari file.

Gradient  : untuk membuat Gradient dalam Text Box.

Textuter : untuk mengisi Text Box dengan Texture yang disediakan oleh Microsoft Word.

6. Untuk mendesain garis/Shape pada Text Box gunakan Menu Shape Outline dan terdiri dari menu :

Untuk memberi warna cukup klik warna yang sudah tersedia.

No Outline : Untuk tidak memberi garis pada Text Box.

Weight : Untuk memilih ketebalan Text Box.

Dashes : Untuk membuat garis pada Text Box putus-putus.

Pattern : Untuk membuat garis pada Text Box dengan Pattern yang disediakan oleh Microsoft word.

Change Shape : Untuk mengganti bentuk-bentuk Text Box sesuai dengan selera.

27. Jelaskan perbedaan antara ComboBox dan TextBox ketika digunakan untuk menerima masukan. Jelaskan kondisi saat TextBox dan ComboBox harus digunakan.​


Jawaban dan Penjelasan:

Perbedaan ComboBox dan EditBox

ComboBox adalah elemen/widget antarmuka pengguna pada aplikasi berbasis GUI (Graphical User Interface), yang merupakan kombinasi (combo = combination) dari dua elemen, yaitu drop-down list (atau list box) dan text box (atau edit box) berbaris tunggal. Dengan ComboBox, pengguna aplikasi dapat memilih satu di antara beberapa pilihan, atau mengetikkan/memasukkan respon seperti halnya pada text box, jika tidak ada opsi pilihan pada ComboBox yang tepat sesuai pilihan pengguna.

Sedangkan TextBox adalah elemen elemen/widget antarmuka pengguna pada aplikasi berbasis GUI (Graphical User Interface), yang berguna untuk menampilkan informasi (jika bersifat read-only), dan menerima respon pengguna dalam bentuk string (kumpulan karakter, atau teks) yang diketikkan. TextBox bisa berbaris tunggal, atau multi-baris (multiline).

Jadi, sebagai rangkuman, dapat kita katakan bahwa:
ComboBox = ListBox + single-line TextBox.

Kondisi Penggunaan ComboBox dan EditBox

Dari perbedaan kedua elemen UI ini, jelas bahwa ComboBox digunakan unuk memberikan beberapa opsi pilihan respon pengguna, yang memungkinkan pengguna memasukkan respon secara manual, jika opsi pilihan yang ingin dipilih/dikirimkan tidak terdapat pada opsi pilihan yang disediakan.

Sedangkan TextBox digunakan ketika aplikasi memerlukan jawaban/respon dari pengguna, yang harus melalui pengetikan secara manual oleh pengguna

Contoh PenggunaanComboBox digunakan untuk menanyakan kepada pengguna tentang warna kesukaan. Warna-warna yang didaftarkan pada ComboBox misalnya adalah merah, kuning, hijau, biru, hitam, dan putih. Karena terdapat banyak warna, dengan ComboBox, pengguna dapat mengirim respon berupa warna lain yang tidak terdapat dalam daftar warna, misalnya dengan mengetikkan "abu-abu".TextBox digunakan untuk menanyakan nama panggilan, nama lengkap pengguna, dan alamat.
