3. kelas 3 SD kak gampang cuman 1 soalmakasih kakkINI SOAL TERAKHIR JANGAN NGASAL PLSSSSini kelas 3 SD tapi soal nya kaya kelas 3 SMP​


Jawab:

Mencari luas Persegi

        s = 14 cm

        L₁ = s x s

            = 14 x 14 = 196 cm²

Mencari Luas Setengah Lingkaran

        d = 14 cm

        r = d : 2 = 14 : 2 = 7 cm

        π = [tex]\frac{22}{7}[/tex]

        L₂ = [tex]\frac{1}{2}[/tex] x π x r²

            = [tex]\frac{1}{2}[/tex] x [tex]\frac{22}{7}[/tex] x 7 x 7

            = 77 cm²

Luas Bangun

        L = L₁ + L₂

           = 196 + 77

           = 273 cm²

Dik:

Lingkaran R = 10 cm

Dit:

Keliling

Jawab:

D = r x 2

D = 10 x 2
