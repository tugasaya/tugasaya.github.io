4. penulisan Bahasa Korea dari Januari hingga Desember thanks ​


Bulan dalam “kalender korea” disebut 월.

Pembahasan

Sistem penanggalan dalam bahasa Korea disebut 달력 (dallyeog) artinya Kalender. Didalam kalender terdapat tiga istilah yaitu “Hari” atau 일 (il), “Bulan” atau 월 (wol), dan “Tahun” atau 년 (nyeon).

Dalam istilah “bulan” terdapat 12 jenisnya, kita pasti sudah mengetahuinya dalam bahasa Indonesia, namun berikut dalam bahasa Korea :

Rumus: Angka Sino Satuan/Belasan + 월

Januari = 일 + 월 (il + wol) => 일월.

Februari = 이 + 월 (i + wol) => 이월.

Maret = 삼 + 월 (sam + wol) => 삼월.

April = 사 + 월 (sa + wol) => 사월.

Mei = 오 + 월 (o + wol) => 오월.

Juni = 유 + 월 (yu + wol) => 유월.

Juli = 칠 + 월 (chil + wol) => 칠월.

Agustus = 팔 + 월 (phal + wol) => 팔월.

Sept. = 구 + 월 (gu + wol) => 구월.

Okt. = 시 + 월 (si - wol) => 시월.

Nov. = 십일 + 월 (sib'il + wol) => 십일월.

Des. = 십이 + 월 (sib'i + wol) => 십이월.

Detail Jawaban

Materi : Sistem Penanggalan

Kata kunci : Nama² bulan didalam bahasa Korea dan penjelasannya.

Kode soal : 18

Kode Kategorisasi : 12.18

23. bulan januari sampai desember disebut bulan


Jawaban:

Hari pertama bulan ini dikenal sebagai Hari Tahun Baru. Bulan ini, rata-rata, merupakan bulan terdingin sepanjang tahun dalam sebagian besar Belahan bumi utara (di mana bulan ini adalah bulan kedua musim dingin) dan bulan terpanas sepanjang tahun dalam sebagian besar Belahan bumi Selatan (di mana bulan ini adalah bulan kedua musim panas). Di belahan bumi Selatan, Januari adalah setara dengan musiman pada bulan Juli pada belahan bumi Utara, dan sebaliknya.
