24. quiz:³√1728 + 8²story an me:ketika sy dekat sama kakak online . Lucu y​


Jawaban :

76

Pembahasan :

Diketahui :

³✓1728 + 8²

Ditanya :

³✓1728 + 8² =...?

Penyelesaian :

Cara pertama :

³✓1728 = 12

8² = 8 × 8 = 64

Maka,

12 + 64 = 76

Cara kedua :

³✓1728 + 8² = 12 + 64

= 76

Kesimpulan :

Jadi, ³✓1728 + 8² = 76

Detail :

Kelas : 5

Mapel : Matematika

Materi : Bangun Ruang

Kata kunci : Pangkat 2 dan akar pangkat 2, pangkat 3 dan akar pangkat 3

#Ayo Belajar#

³√1728 + 8²

= [tex]\sqrt[3]{12^{3} }[/tex] + 8 × 8

= [tex]12^{\frac{3}{3} }[/tex] + 64

= [tex]12^{1}[/tex] + 64

= 12 + 64

= 76
