5. carilah cara menonaktifkan sistem windows 7,8,9,10​


Jawaban:

Untuk menonaktifkan sistem Windows, Anda dapat melakukan hal-hal berikut:

Melalui Control Panel: Buka Control Panel dan pilih "System and Security." Kemudian, pilih "Power Options" dan pilih "Choose what the power buttons do." Di sini, Anda dapat mengatur tombol daya untuk menonaktifkan sistem saat ditekan.

Melalui Command Prompt: Buka Command Prompt dengan hak akses administrator dan ketik "shutdown /s /t 0" tanpa tanda petik. Ini akan menonaktifkan sistem secara instan.

Melalui Shortcut Keyboard: Anda dapat membuat shortcut keyboard untuk menonaktifkan sistem Windows. Caranya adalah dengan membuat shortcut baru di desktop dan menambahkan "shutdown /s /t 0" tanpa tanda petik sebagai target. Kemudian Anda tinggal menekan tombol shortcut yang telah anda buat.

Perlu diingat bahwa menonaktifkan sistem secara tiba-tiba tanpa menyimpan data Anda dapat menyebabkan kerusakan pada data Anda.

16. Jelaskan cara menonaktifkan komputer berbasis Windows


1.Menekan tombol power. Ini adalah cara tercepat untuk mematikan komputer. Tombol power di laptop atau PC jika ditekan akan segera terjadi penutupan beragam aplikasi secara instan, lalu mati. Hanya saja cara ini sebaiknya digunakan ketika benar-benar dalam posisi terpaksa, seperti komputer hang. Pasalnya, langkah ini memungkinkan terjadinya kerusakan sistem file, sistem operasi, hingga program yang saat itu masih terbuka.

2.Menerapkan shortcut combo. Ini dilakukan dengan memencet beberapa tombol yang ada di keyboard. Bahkan, Anda tidak perlu menggunakan campur tangan mouse untuk urusan ini. Pada sistem operasi Windows 7, penerapannya sangat sederhana. Tinggal tekan tombol Windows, tekan panah kanan, lalu tekan tombol Enter. Sementara pada Windows 10 hanya dengan menekan tombol Windows, lalu tekan X,u,u.

3.Membuat shortcut taskbar di laman desktop. Bisa pula mematikan komputer dengan mengeklik shortcut buatan sendiri yang diletakkan di desktop. Cara seperti ini:

Klik kanan pada desktop, tekan New lalu Shortcut.Untuk isian di kota Location, tuliskan perintah ini: “%windir%system32shutdown.exe -s -t 0” (tanpa tanda “)Klik Next, lalau namai shortcut sesuai keinginan.Shortcut yang sudah dibuat dapat diklik saat Anda ingin mematikan komputer.

22. Bagaimana cara menonaktifkan hak akses yang telah diberikan di OneDrive?( no ngasal ) ​


Jawaban:

Pilih file atau folder yang ingin dihentikan pembagiannya. Pilih Informasi Informasi di sudut kanan atas untuk membuka panel Detail.

Catatan: (Ikon informasi berada tepat di bawah gambar profil Anda. Jika Anda tidak melihatnya, pastikan Anda hanya memiliki satu file atau folder yang dipilih-tidak akan ditampilkan jika Anda memilih beberapa item.)

Pada panel detail , di bawah header has Access , Anda akan melihat ikon orang , ikon link , dan/atau ikon email . Opsi ini berbeda-beda tergantung pada cara Anda berbagi file atau folder. Pilih Kelola akses dan:

Untuk berhenti berbagi file seluruhnya, klik Hentikan berbagi.

Untuk menghapus link berbagi, klik tautan ... di samping tautan, lalu klik X di samping tautan untuk menghapusnya. (Anda akan mendapatkan perintah yang menanyakan apakah Anda yakin ingin menghapus tautan-jika demikian, klik Hapus tautan.)

Untuk berhenti berbagi dengan orang tertentu, Perluas daftar di bawah link orang tertentu dan klik X untuk menghapus seseorang.  Untuk mengubah izin atau berhenti berbagi dengan seseorang yang memiliki akses langsung, di bawah akses langsung, klik menu menurun di samping nama orang tersebut dan pilih opsi yang Anda inginkan.

24. apa maksud g di keyboard? cara menonaktifkannya gimana?


Bagi Anda pengguna komputer maupun laptop pasti sudah sering menggunakan tombol CTRL. Terlebih bagi yang sering melakukan copy paste dengan CTRL+C dan CTRL +V.
Namun bukan hanya huruf C dan V saja yang mempunyai fungsi jika digabungkan dengan tombol CTRL. Berikut ini fungsi-fungsinya CTRL + A to Z :
Ctrl + A = Memblok/menyeleksi seluruh dokumen (All)Ctrl + B = Memberikan efek tebal pada huruf (Bold)Ctrl + C = Menyalin dokumen (Copy)Ctrl + D = Menggandakan data atau gambar (Double)Ctrl + E = Mengetengahkan documentCtrl + F = Mencari satu kata atau kalimat (Find)Ctrl + G = Tabulasi pada Open OfficeCtrl + H = Mengaktifkan kotak History pada browserCtrl + I = Memiringkan tulisan (Italic)Ctrl + J = Mengaktifkan kotak download pada browserCtrl + K =Mencari cepat dengan Mozilla pada kotak search barCtrl + L = Memasukkan alamat web pada browser (Location)Ctrl + M = Mengaktifkan email client pada OperaCtrl + N = Membuka lembar baru (New)Ctrl + O = Membuka dokumen (Open)Ctrl + P = Mencetak halaman (Print)Ctrl + Q = Keluar secara cepat untuk Open Office (Quit)Ctrl + R = Meratakan dokumen sebelah kanan (Right)Ctrl + S = Menyimpan dokumen (Save)Ctrl + T = Membuka tab baru pada browser (Tab)Ctrl + U = Memberikan garis bawah pada kata (Open Office, Word, dan lain-lain) (Underline)Ctrl + V = Menempelkan kata yang sudah di copy (Paste)Ctrl + W = Menutup tab browserCtrl + X = Memindahkan kata (Cut)Ctrl + Y = Mengembalikan apa yang sudah di Undo (Redo)Ctrl + Z = Mengembalikan ke keadaan seperti sebelumnya. (Undo)
